# 更新日志

## v2.0 (2026-02-06)

### 重大更新

#### 数据源全面升级
- **AKShare 作为主力数据源**：提供完整、准确、免费的市场数据
- **智能数据源切换**：AKShare → 开盘啦 → TuShare 多级降级策略
- **TuShare 变为可选**：不配置也能正常运行

#### 数据完整性修复
- ✅ **涨停数据完整**：获取全部涨停个股（包括首板和连板）
  - 修复前：仅获取 9 只连板股票
  - 修复后：获取 44 只涨停股票（34只首板 + 10只连板）
  
- ✅ **市值数据准确**：直接使用 AKShare 提供的市值数据
  - 总市值：精确到元，转换为亿元
  - 流通市值：精确到元，转换为亿元
  - 不再使用估算值

- ✅ **融资融券优化**：改用 AKShare 免费接口
  - 修复前：使用 TuShare 接口，需要 5000+ 积分
  - 修复后：使用 AKShare 接口，免费获取 1800+ 标的

#### 功能优化
- ✅ **LLM 解析稳定性**：修复 Prompt 模板和 JSON 解析逻辑
- ✅ **空数据友好提示**：当无显著特征时显示"今日无xxx"
- ✅ **连板数据准确性**：使用 AKShare 提供的准确连板数

#### 代码清理
- 删除 18 个临时测试文件和文档
- 保留核心功能文件和必要文档
- 优化项目结构

### 技术细节

#### 数据源集成
```python
# kaipanla_source.py
- 集成 AKShare 的 stock_zt_pool_em 接口
- 获取完整涨停数据（首板 + 连板）
- 提取市值、成交额、连板数等字段

# data_source_adapter.py
- 优先初始化 AKShare
- TuShare 作为辅助数据源
- 融资融券强制使用 AKShare

# tushare_source.py
- 接受 akshare_source 参数
- 融资融券优先使用 AKShare
- 市值数据优先使用 AKShare
```

#### 数据字段映射
| 字段 | AKShare | 单位转换 |
|------|---------|----------|
| 总市值 | 总市值 | 元 → 亿元 (÷1e8) |
| 流通市值 | 流通市值 | 元 → 亿元 (÷1e8) |
| 成交额 | 成交额 | 元 → 亿元 (÷1e8) |
| 连板数 | 连板数 | 直接使用 |
| 封板时间 | 首次封板时间 | HHMMSS → HH:MM:SS |

### 升级指南

#### 从 v1.0 升级到 v2.0

1. **更新依赖**
   ```bash
   pip install --upgrade akshare
   ```

2. **配置调整**（可选）
   - TuShare Token 不再必需
   - 如需更详细的基础信息，可继续配置 TuShare

3. **运行测试**
   ```bash
   python run_analysis.py
   ```

4. **验证数据**
   - 检查涨停个股数量是否完整
   - 检查市值数据是否准确
   - 检查融资融券标的是否正常
