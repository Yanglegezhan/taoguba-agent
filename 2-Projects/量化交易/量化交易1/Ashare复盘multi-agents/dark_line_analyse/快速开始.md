# Dark Line Analyse - 快速开始

## 🚀 5分钟上手

### 1. 安装依赖

```bash
pip install tushare akshare pandas numpy
```

### 2. 运行程序

#### 分析今天（推荐）

```bash
python dark_line_cli.py
```

**效果**：
- ✅ 自动使用实时接口
- ✅ 获取完整涨停数据（首板+连板）
- ✅ 生成详细分析报告

#### 分析历史

```bash
python dark_line_cli.py -d 2026-01-23
```

**说明**：
- ⚠️  仅获取连板股票（API限制）
- ⚠️  数据量约10-20%

### 3. 查看报告

报告保存在：
```
output/reports/dark_line_analysis_YYYYMMDD.json
output/reports/dark_line_analysis_YYYYMMDD.md
```

## 📊 数据源配置（可选）

### 方案1：使用 Tushare（推荐）

**优点**：数据最完整

**步骤**：
1. 访问 https://tushare.pro/ 注册
2. 获取 API Token（需要2000积分）
3. 编辑 `config/config.yaml`：

```yaml
data_source:
  tushare:
    api_token: "你的Token"
```

### 方案2：使用 AKShare（免费）

**优点**：完全免费

**步骤**：
1. 确保 `config/config.yaml` 中 Tushare token 为空
2. 程序会自动使用 AKShare

```yaml
data_source:
  tushare:
    api_token: ""  # 留空
```

### 方案3：仅使用开盘啦（默认）

**特点**：无需配置，自动降级

## ❓ 常见问题

### Q1: 为什么历史数据只有几只股票？

**答**：这是开盘啦API的限制，历史接口只返回连板股票。

**解决**：分析今天的数据（使用实时接口，完整数据）

### Q2: 如何获取完整的历史数据？

**答**：每日盘后运行程序并保存数据

### Q3: Tushare Token 在哪里获取？

**答**：访问 https://tushare.pro/ 注册并获取（需要2000积分）

### Q4: AKShare 访问超时怎么办？

**答**：程序已自动禁用代理。如果仍有问题，配置 Tushare Token

## 🎯 推荐工作流

### 每日分析流程

1. **盘后运行**（15:30后）
   ```bash
   python dark_line_cli.py
   ```

2. **查看报告**
   ```
   output/reports/dark_line_analysis_YYYYMMDD.md
   ```

3. **分析暗线**
   - 查看检测到的暗线
   - 分析统计特征
   - 研究命名规律

4. **保存数据**（可选）
   - 将报告归档
   - 建立数据库

### 历史回测流程

1. **选择日期**
   ```bash
   python dark_line_cli.py -d 2026-01-23
   ```

2. **分析连板股票**
   - 专注于连板股票的暗线
   - 研究连板规律

3. **对比分析**
   - 对比不同日期的暗线
   - 寻找规律

## 🔧 高级配置

### 修改超时时间

编辑 `config/config.yaml`：

```yaml
data_source:
  kaipanla:
    timeout: 120  # 增加到120秒
```

### 修改分析参数

```yaml
analysis:
  lift_ratio_threshold: 2.0  # 提升比阈值
  min_sample_size: 3  # 最小样本数
  naming_similarity_threshold: 0.7  # 命名相似度阈值
```
