## ✨ 新增功能

### 1. 板块资金成交额数据获取 (`get_sector_capital_data`)

#### 功能描述
获取指定板块的资金成交额数据，包括成交额、主力资金流向、市值等14个关键指标。

#### API接口
```
POST https://apphwhq.longhuvip.com/w1/api/index.php
参数: a=GetPanKou&c=ZhiShuL2Data&StockID={板块代码}&Day={日期}
```

#### 返回数据字段（14个）
| 字段 | 说明 | 单位 |
|------|------|------|
| turnover | 成交额 | 元 |
| change_pct | 涨跌幅 | % |
| market_cap | 市值 | 亿元 |
| main_net_inflow | 主力净额 | 元 |
| main_sell | 主卖 | 元 |
| net_amount | 净额 | 元 |
| up_count | 上涨家数 | 只 |
| down_count | 下跌家数 | 只 |
| flat_count | 平盘家数 | 只 |
| circulating_market_cap | 流通市值 | 元 |
| total_market_cap | 总市值 | 元 |
| turnover_rate | 换手率 | % |
| main_net_inflow_pct | 主力净占比 | % |

#### 使用示例
```python
from kaipanla_crawler import KaipanlaCrawler

crawler = KaipanlaCrawler()

# 获取化工板块实时数据
data = crawler.get_sector_capital_data("801235")
print(f"成交额: {data['turnover'] / 100000000:.2f}亿元")
print(f"主力净额: {data['main_net_inflow'] / 100000000:.2f}亿元")

# 获取历史数据
data = crawler.get_sector_capital_data("801235", "2026-01-20")
```

### 2. N日板块强度排名 (`get_sector_strength_ndays`)

#### 功能描述
获取N个交易日的板块强度排名数据，支持分析板块热度趋势和轮动。

#### 实现方式
基于现有的 `get_sector_ranking()` 函数扩展，循环获取多日数据并汇总。

#### 返回数据字段（5个）
| 字段 | 说明 |
|------|------|
| 日期 | 交易日期 |
| 板块代码 | 板块代码 |
| 板块名称 | 板块名称 |
| 涨停数 | 该板块涨停股票数量 |
| 涨停股票 | 涨停股票代码列表（逗号分隔） |

#### 使用示例
```python
from kaipanla_crawler import KaipanlaCrawler

crawler = KaipanlaCrawler()

# 获取7日板块强度数据
df = crawler.get_sector_strength_ndays("2026-01-20", num_days=7)

# 分析板块热度趋势
sector_trend = df.groupby('板块名称')['涨停数'].sum().sort_values(ascending=False)
print("7日最强板块:")
print(sector_trend.head(10))

# 查看特定板块的每日涨停数
sector_data = df[df['板块名称'] == '化工']
print(sector_data[['日期', '涨停数']])
```

## 🎯 应用场景

### 1. 板块资金监控
实时监控多个板块的资金流向，识别主力资金动向。

```python
# 监控多个板块
sectors = {"801235": "化工", "801346": "电力设备", "801225": "机械设备"}
for sector_code, sector_name in sectors.items():
    data = crawler.get_sector_capital_data(sector_code)
    print(f"{sector_name}: 主力净额 {data['main_net_inflow']/100000000:.2f}亿")
```

### 2. 板块强度分析
分析最近N日的板块强度，找出持续强势的板块。

```python
# 获取7日数据
df = crawler.get_sector_strength_ndays("2026-01-20", num_days=7)

# 计算持续性
sector_persistence = df[df['涨停数'] > 0].groupby('板块名称').size()
print("持续强势板块:", sector_persistence.head(10))
```

### 3. 板块轮动识别
对比不同时间段的板块强度，识别轮动机会。

```python
# 对比最近3日和前4日
dates = sorted(df['日期'].unique(), reverse=True)
recent_3days = dates[:3]
previous_4days = dates[3:7]

recent_strength = df[df['日期'].isin(recent_3days)].groupby('板块名称')['涨停数'].sum()
previous_strength = df[df['日期'].isin(previous_4days)].groupby('板块名称')['涨停数'].sum()

strength_change = recent_strength - previous_strength
print("强度上升的板块:", strength_change.sort_values(ascending=False).head(5))
```

### 4. 资金+强度综合分析
结合资金数据和强度数据，进行综合评估。

```python
# 获取最强板块
sector_trend = df.groupby('板块名称')['涨停数'].sum().sort_values(ascending=False)
top_sector = sector_trend.index[0]

# 获取该板块的资金数据
sector_code = df[df['板块名称'] == top_sector].iloc[0]['板块代码']
capital_data = crawler.get_sector_capital_data(sector_code)

# 综合评分
score = (sector_trend[top_sector] * 0.4 + 
         (capital_data['main_net_inflow'] / 1000000000) * 0.3 + 
         (capital_data['turnover'] / 100000000000) * 0.3)
```

## 🔧 技术细节

### API数据格式
板块资金数据的pankou数组格式：
```python
[
    成交额,           # pankou[0] - 元
    涨跌幅,           # pankou[1] - %
    市值,            # pankou[2] - 亿元
    主力净额,         # pankou[3] - 元
    主卖,            # pankou[4] - 元
    净额,            # pankou[5] - 元
    上涨家数,         # pankou[6] - 只
    下跌家数,         # pankou[7] - 只
    平盘家数,         # pankou[8] - 只
    流通市值,         # pankou[9] - 元
    总市值,          # pankou[10] - 元
    换手率           # pankou[11] - %
]
```

### 交易日处理
`get_sector_strength_ndays()` 自动过滤非交易日：
- 向前推算 `num_days * 2` 个日期
- 请求每个日期的数据
- 如果返回空数据，跳过（非交易日）
- 直到获取到 `num_days` 个交易日数据

### 性能优化
- 使用 `timeout` 参数控制超时时间（默认60秒）
- 支持批量获取多日数据
- 返回 DataFrame 格式便于分析

## ⚠️ 注意事项

1. **板块代码**: 需要使用正确的板块代码，可以从开盘啦APP或 `get_sector_ranking()` 获取
2. **日期格式**: 必须是 `YYYY-MM-DD` 格式
3. **交易日**: 只能查询交易日数据，非交易日会自动跳过
4. **超时设置**: 默认60秒，网络不稳定时可以增加
5. **实时数据**: `get_sector_capital_data()` 不传date参数获取实时数据
6. **数据来源**: 数据来自开盘啦APP，准确率99%+
