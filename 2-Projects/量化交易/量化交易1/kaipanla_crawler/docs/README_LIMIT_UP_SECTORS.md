# 涨停原因板块爬虫功能

## 功能说明

`get_limit_up_sectors()` 函数用于获取开盘啦APP的"涨停原因板块"数据，返回当日各板块的涨停股票详情。

## 使用方法

```python
from kaipanla_crawler_new import KaipanlaCrawler

crawler = KaipanlaCrawler()

# 获取指定日期的涨停原因板块数据
data = crawler.get_limit_up_sectors("2026-01-16")

# 访问市场概况
print(data['summary'])
# 输出：
# {
#     "日期": "2026-01-16",
#     "上涨家数": 2251,
#     "下跌家数": 2807,
#     "涨停数": 58,
#     "跌停数": 46,
#     "涨跌比": 40.21,
#     "昨日涨跌比": -0.32
# }

# 遍历板块
for sector in data['sectors']:
    print(f"板块: {sector['sector_name']}")
    print(f"涨停股票数: {sector['stock_count']}")
    
    # 遍历该板块的涨停股票
    for stock in sector['stocks']:
        print(f"  {stock['股票代码']} {stock['股票名称']}")
        print(f"  涨停价: {stock['涨停价']}")
        print(f"  连板: {stock['连板描述']}")
        print(f"  概念: {stock['概念标签']}")
        print(f"  主题: {stock['主题']}")
        print(f"  涨停原因: {stock['涨停原因']}")
```

## 字段说明

### summary（市场概况）
- `日期`: 数据日期
- `上涨家数`: 当日上涨股票数量
- `下跌家数`: 当日下跌股票数量
- `涨停数`: 当日涨停股票数量
- `跌停数`: 当日跌停股票数量
- `涨跌比`: 涨跌比例
- `昨日涨跌比`: 昨日涨跌比例

### sectors（板块列表）
- `sector_code`: 板块代码
- `sector_name`: 板块名称
- `stock_count`: 该板块涨停股票数量
- `stocks`: 涨停股票列表

### stocks（股票详情）
- `股票代码`: 股票代码
- `股票名称`: 股票名称
- `涨停价`: 涨停价格
- `成交额`: 成交额（元）
- `连板天数`: 连板天数
- `封单额`: 封单金额（元）
- `连板描述`: 连板描述（如"2连板"、"首板"）
- `连板次数`: 连板次数
- `概念标签`: 相关概念标签
- `主力资金`: 主力资金（元）
- `总市值`: 总市值（元）
- `涨停时间`: 涨停时间（如28.2表示9:28:12）
- `流通市值`: 流通市值（元）
- `主题`: 涨停主题
- `涨停原因`: 详细涨停原因说明
- `是否首板`: 是否首板（1=是，0=否）

## 参数说明

- `date`: 日期，格式为"YYYY-MM-DD"，默认为当前日期
- `index`: 分页索引，默认为0（第一页）

## 注意事项

1. 该接口返回的是字典格式，不是DataFrame
2. 如需DataFrame格式的板块数据，请使用 `get_sector_ranking()` 函数
3. 数据来源于开盘啦APP的"涨停原因板块"功能
4. 建议在交易日收盘后使用，数据更完整
