# 多头空头风向标功能实现总结

## 功能概述

实现了 `get_sentiment_indicator()` 函数，用于获取市场多头空头风向标数据，通过分析特定板块的股票排序，识别市场情绪的多空方向。

## 实现时间

2026-01-19

## API接口

- **接口名称**: `PlateIntroduction_Info`
- **请求方式**: POST
- **接口地址**: `https://apphwhq.longhuvip.com/w1/api/index.php`
- **数据类型**: 实时数据

## 函数签名

```python
def get_sentiment_indicator(self, plate_id="801225", stocks=None)
```

## 参数说明

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| plate_id | str | 否 | "801225" | 板块ID |
| stocks | list | 否 | None | 股票代码列表，不提供则使用默认列表（27只） |

## 返回值结构

```python
{
    "date": "2026-01-19",              # 日期
    "plate_id": "801225",              # 板块ID
    "bullish_codes": [                 # 多头风向标（前3只）
        "002112",
        "603667",
        "600550"
    ],
    "bearish_codes": [                 # 空头风向标（后3只）
        "001255",
        "000681",
        "002465"
    ],
    "all_stocks": [                    # 所有股票代码列表
        "002112", "603667", "600550", ...
    ]
}
```

## 核心功能

1. **多头风向标识别**
   - 自动提取股票列表中排名前3的股票
   - 代表市场多头力量

2. **空头风向标识别**
   - 自动提取股票列表中排名后3的股票
   - 代表市场空头力量

3. **自定义股票列表**
   - 支持传入自定义股票列表
   - 灵活适应不同分析需求

4. **实时数据**
   - 获取最新的市场情绪数据
   - 无需传入日期参数

## 实现细节

### 1. API请求

```python
data = {
    "a": "PlateIntroduction_Info",
    "c": "ZhiShuRanking",
    "PhoneOSNew": "1",
    "Stocks": stocks_str,              # 股票代码，逗号分隔
    "DeviceID": str(uuid.uuid4()),
    "VerSion": "5.21.0.2",
    "apiv": "w42",
    "PlateID": plate_id
}
```

### 2. 数据解析

- 从API返回的 `List` 字段中提取股票代码
- 如果API未返回数据，使用输入的股票列表作为备选
- 前3只作为多头风向标
- 后3只作为空头风向标

### 3. 错误处理

- 网络请求失败时返回空列表
- API返回错误时打印错误信息并返回空列表
- 数据解析失败时使用输入的股票列表

## 使用示例
