## 用户需求回顾

用户提供了以下信息：

1. **API请求**:
   ```
   POST https://apphis.longhuvip.com/w1/api/index.php
   参数: a=GetDayNewHigh_W28&st=360&c=StockNewHigh&...
   ```

2. **返回数据格式**:
   ```json
   {"x": ["20240725_61_18_0", "20240726_58_11_0", ...]}
   ```

3. **字段说明**:
   - `20260116`: 日期
   - `478`: 新高数量
   - `127`: 今日新增（需要提取的字段）
   - `0`: 保留字段

4. **功能要求**:
   - 输入: 起始日期和结束日期
   - 输出: Series格式，索引为日期，内容为今日新增
   - 只提取"今日新增"字段

## 实现方案

### 函数签名

```python
def get_new_high_data(self, end_date, start_date=None):
    """
    获取百日新高数据
    
    Args:
        end_date: 结束日期，格式YYYY-MM-DD
        start_date: 起始日期，格式YYYY-MM-DD，可选
        
    Returns:
        pd.Series: 索引为日期，值为今日新增新高数量
    """
```

### 核心实现

1. **API请求**: 使用POST方法请求数据
2. **数据解析**: 从返回的`x`字段中提取数据
3. **字段提取**: 使用`split("_")`分割，取第3个字段（索引2）
4. **日期转换**: 将`20260116`转换为`2026-01-16`
5. **返回处理**: 
   - 单日查询返回数值
   - 范围查询返回Series

## 使用示例

### 基本用法

```python
from kaipanla_crawler import KaipanlaCrawler

crawler = KaipanlaCrawler()

# 获取单日数据
count = crawler.get_new_high_data("2026-01-16")
print(f"今日新增: {count} 只")  # 输出: 今日新增: 127 只

# 获取日期范围数据
```
