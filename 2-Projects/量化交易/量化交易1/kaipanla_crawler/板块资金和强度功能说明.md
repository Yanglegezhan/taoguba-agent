# 板块资金和强度功能说明

## 功能概述

新增两个板块分析功能：
1. **get_sector_capital_data()** - 获取板块资金成交额数据
2. **get_sector_strength_ndays()** - 获取N日板块强度排名数据

---

## 1. 板块资金成交额数据

### 功能说明

`get_sector_capital_data()` 函数用于获取指定板块的资金成交额数据，包括成交额、主力资金流向、市值等关键指标。

### 函数签名

```python
def get_sector_capital_data(self, sector_code, date=None, timeout=60):
    """
    获取板块资金成交额数据
    
    Args:
        sector_code: 板块代码，如 "801235"（化工）
        date: 日期，格式YYYY-MM-DD，默认为空（获取实时数据）
        timeout: 超时时间（秒），默认60秒
        
    Returns:
        dict: 包含板块资金数据
    """
```

### 返回数据字段

| 字段 | 类型 | 说明 | 单位 |
|------|------|------|------|
| sector_code | str | 板块代码 | - |
| date | str | 日期 | YYYY-MM-DD |
| turnover | float | 成交额 | 元 |
| change_pct | float | 涨跌幅 | % |
| market_cap | float | 市值 | 亿元 |
| main_net_inflow | float | 主力净额 | 元 |
| main_sell | float | 主卖 | 元 |
| net_amount | float | 净额 | 元 |
| up_count | int | 上涨家数 | 只 |
| down_count | int | 下跌家数 | 只 |
| flat_count | int | 平盘家数 | 只 |
| circulating_market_cap | float | 流通市值 | 元 |
| total_market_cap | float | 总市值 | 元 |
| turnover_rate | float | 换手率 | % |
| main_net_inflow_pct | float | 主力净占比 | % |

### 使用示例

#### 示例1: 获取板块实时资金数据

```python
from kaipanla_crawler import KaipanlaCrawler

crawler = KaipanlaCrawler()

# 获取化工板块（801235）实时数据
data = crawler.get_sector_capital_data("801235")

print(f"板块代码: {data['sector_code']}")
print(f"成交额: {data['turnover'] / 100000000:.2f}亿元")
print(f"涨跌幅: {data['change_pct']:.2f}%")
print(f"主力净额: {data['main_net_inflow'] / 100000000:.2f}亿元")
print(f"主力净占比: {data['main_net_inflow_pct']:.2f}%")
print(f"上涨家数: {data['up_count']}")
print(f"下跌家数: {data['down_count']}")
```

#### 示例4: 计算板块资金强度

```python
from kaipanla_crawler import KaipanlaCrawler

crawler = KaipanlaCrawler()

sector_code = "801235"
data = crawler.get_sector_capital_data(sector_code)

# 计算资金强度指标
total_stocks = data['up_count'] + data['down_count'] + data['flat_count']
up_ratio = data['up_count'] / total_stocks if total_stocks > 0 else 0

print(f"板块资金强度分析:")
print(f"  成交额: {data['turnover'] / 100000000:.2f}亿元")
print(f"  主力净额: {data['main_net_inflow'] / 100000000:.2f}亿元")
print(f"  主力净占比: {data['main_net_inflow_pct']:.2f}%")
print(f"  上涨比例: {up_ratio * 100:.2f}%")
print(f"  换手率: {data['turnover_rate']:.2f}%")

# 判断资金强度
if data['main_net_inflow'] > 0 and up_ratio > 0.6:
    print("\n✓ 资金强度: 强势（主力净流入 + 上涨比例高）")
elif data['main_net_inflow'] < 0 and up_ratio < 0.4:
    print("\n✗ 资金强度: 弱势（主力净流出 + 上涨比例低）")
else:
    print("\n- 资金强度: 中性")
```

## 2. N日板块强度排名

### 功能说明

`get_sector_strength_ndays()` 函数用于获取N个交易日的板块强度排名数据，支持分析板块热度趋势。

### 函数签名

```python
def get_sector_strength_ndays(self, end_date, num_days=7, timeout=60):
    """
    获取N日板块强度排名数据
    
    Args:
        end_date: 结束日期，格式YYYY-MM-DD
        num_days: 获取天数，默认7天
        timeout: 超时时间（秒），默认60秒
        
    Returns:
        pd.DataFrame: 包含N日板块强度数据
    """
```

### 返回数据字段

| 字段 | 类型 | 说明 |
|------|------|------|
| 日期 | str | 交易日期 |
| 板块代码 | str | 板块代码 |
| 板块名称 | str | 板块名称 |
| 涨停数 | int | 该板块涨停股票数量 |
| 涨停股票 | str | 涨停股票代码列表（逗号分隔） |

### 使用示例

#### 示例1: 获取7日板块强度数据

from kaipanla_crawler import KaipanlaCrawler

#### 示例5: 识别板块轮动

```python
from kaipanla_crawler import KaipanlaCrawler

crawler = KaipanlaCrawler()

df = crawler.get_sector_strength_ndays("2026-01-20", num_days=7)

# 计算每个板块每日的涨停数排名
df['排名'] = df.groupby('日期')['涨停数'].rank(ascending=False, method='min')

# 找出排名变化最大的板块
sector_rank_change = df.groupby('板块名称')['排名'].agg(['min', 'max'])
sector_rank_change['变化'] = sector_rank_change['max'] - sector_rank_change['min']
sector_rank_change = sector_rank_change.sort_values('变化', ascending=False)

print("板块排名变化最大的TOP 5（可能存在轮动）:\n")
print(f"{'板块名称':<15} {'最高排名':<10} {'最低排名':<10} {'排名变化':<10}")
print("-" * 50)
for sector_name, row in sector_rank_change.head(5).iterrows():
    print(f"{sector_name:<15} {int(row['min']):<10} {int(row['max']):<10} {int(row['变化']):<10}")
```

## 应用场景

### 场景1: 板块资金监控

```python
# 实时监控多个板块的资金流向
import time

sectors = {
    "801235": "化工",
    "801346": "电力设备",
    "801225": "机械设备"
}

while True:
    print(f"\n[{datetime.now().strftime('%H:%M:%S')}] 板块资金监控")
    print("-" * 60)
    
    for sector_code, sector_name in sectors.items():
        data = crawler.get_sector_capital_data(sector_code)
        if data:
            print(f"{sector_name}: "
                  f"成交额 {data['turnover']/100000000:.2f}亿 | "
                  f"涨跌幅 {data['change_pct']:.2f}% | "
                  f"主力净额 {data['main_net_inflow']/100000000:.2f}亿")
    
    time.sleep(300)  # 每5分钟更新一次
```

### 场景2: 板块强度分析

```python
# 分析最近7日的板块强度，找出持续强势的板块
df = crawler.get_sector_strength_ndays("2026-01-20", num_days=7)

# 计算每个板块的持续性（有涨停的天数）
sector_persistence = df[df['涨停数'] > 0].groupby('板块名称').size()
sector_persistence = sector_persistence.sort_values(ascending=False)

print("持续强势板块（7日内有涨停的天数）:\n")
for sector_name, days in sector_persistence.head(10).items():
    total_count = df[df['板块名称'] == sector_name]['涨停数'].sum()
    print(f"{sector_name}: {days}天有涨停，总涨停数 {total_count}")
```

### 场景3: 板块轮动策略

```python
# 识别板块轮动机会
df = crawler.get_sector_strength_ndays("2026-01-20", num_days=7)

# 计算最近3日和前4日的涨停数
dates = sorted(df['日期'].unique(), reverse=True)
recent_3days = dates[:3]
previous_4days = dates[3:7]

recent_strength = df[df['日期'].isin(recent_3days)].groupby('板块名称')['涨停数'].sum()
previous_strength = df[df['日期'].isin(previous_4days)].groupby('板块名称')['涨停数'].sum()

# 找出强度上升的板块
strength_change = recent_strength - previous_strength
rising_sectors = strength_change.sort_values(ascending=False)

print("强度上升的板块（可能的轮动机会）:\n")
for sector_name, change in rising_sectors.head(5).items():
    print(f"{sector_name}: 涨停数增加 {change:.0f}")
```

## 注意事项

1. **板块代码**: 需要使用正确的板块代码（如801235），可以从开盘啦APP获取
2. **日期格式**: 必须是`YYYY-MM-DD`格式
3. **交易日**: 只能查询交易日数据，非交易日会自动跳过
4. **超时设置**: 默认60秒，网络不稳定时可以增加
5. **数据来源**: 数据来自开盘啦APP，准确率99%+
6. **实时数据**: `get_sector_capital_data()`不传date参数获取实时数据
