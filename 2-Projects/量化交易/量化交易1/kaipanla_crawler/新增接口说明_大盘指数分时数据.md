# 新增接口说明：大盘指数分时数据

## 概述

为支持**题材复盘智能体**的盘面联动分析功能，新增了 `get_index_intraday()` 接口，用于获取大盘指数的分时数据。

## 接口信息

### 方法签名
```python
def get_index_intraday(self, index_code="SH000001", date=None, timeout=60)
```

### 参数说明
- **index_code** (str): 指数代码，默认"SH000001"（上证指数）
  - `SH000001`: 上证指数
  - `SZ399001`: 深证成指
  - `SZ399006`: 创业板指
  - `SH000300`: 沪深300
- **date** (str, 可选): 日期，格式YYYY-MM-DD，默认为None（获取实时数据）
- **timeout** (int): 超时时间（秒），默认60秒

### 返回数据结构
```python
{
    "index_code": "SH000001",           # 指数代码
    "index_name": "上证指数",            # 指数名称
    "date": "2026-01-22",               # 日期
    "open": 3234.56,                    # 开盘价
    "close": 3245.67,                   # 收盘价
    "high": 3250.00,                    # 最高价
    "low": 3220.00,                     # 最低价
    "preclose": 3230.00,                # 昨收价
    "change_pct": 0.48,                 # 涨跌幅（%）
    "data": DataFrame                   # 分时数据
}
```

### 分时数据DataFrame字段
| 字段 | 类型 | 说明 |
|------|------|------|
| time | str | 时间（HH:MM格式） |
| timestamp | datetime | 时间戳对象 |
| price | float | 价格 |
| pct_change | float | 涨跌幅（相对昨收，%） |
| volume | int | 成交量（手） |
| turnover | float | 成交额（元） |

## 使用示例

### 1. 获取实时分时数据
```python
from kaipanla_crawler import KaipanlaCrawler

crawler = KaipanlaCrawler()

# 获取上证指数实时分时数据
data = crawler.get_index_intraday("SH000001")

print(f"指数: {data['index_name']}")
print(f"当前价: {data['close']:.2f}")
print(f"涨跌幅: {data['change_pct']:.2f}%")

# 查看分时数据
df = data['data']
print(df.head())
```

### 2. 获取历史分时数据
```python
# 获取指定日期的分时数据
data = crawler.get_index_intraday("SH000001", "2026-01-20")

print(f"日期: {data['date']}")
print(f"收盘价: {data['close']:.2f}")
print(f"涨跌幅: {data['change_pct']:.2f}%")
```

### 3. 识别关键变盘点（用于盘面联动分析）
```python
data = crawler.get_index_intraday("SH000001")
df = data['data']

# 找到最低点（急跌低点）
min_idx = df['pct_change'].idxmin()
min_point = df.loc[min_idx]
print(f"最低点时间: {min_point['time']}, 涨跌幅: {min_point['pct_change']:.2f}%")

# 找到最高点（突破点）
max_idx = df['pct_change'].idxmax()
max_point = df.loc[max_idx]
print(f"最高点时间: {max_point['time']}, 涨跌幅: {max_point['pct_change']:.2f}%")

# 识别V型反转
if min_idx < len(df) - 10:
    after_min = df.iloc[min_idx:min_idx+10]
    rebound = after_min['pct_change'].max() - min_point['pct_change']
    
    if rebound > 0.5:  # 反弹超过0.5%
        print(f"✓ 检测到V型反转，反弹幅度: {rebound:.2f}%")
```

### 4. 计算板块与大盘的时差
```python
# 获取大盘分时数据
index_data = crawler.get_index_intraday("SH000001")
index_df = index_data['data']

# 获取板块分时数据
sector_data = crawler.get_sector_intraday("801235")  # 化工板块
sector_df = sector_data['data']

# 找到大盘最低点
index_min_idx = index_df['pct_change'].idxmin()
index_min_time = index_df.loc[index_min_idx, 'time']

# 找到板块最低点
sector_min_idx = sector_df['pct_change'].idxmin()
sector_min_time = sector_df.loc[sector_min_idx, 'time']

print(f"大盘最低点时间: {index_min_time}")
print(f"板块最低点时间: {sector_min_time}")

# 计算时差（简化版，实际需要转换为分钟）
if sector_min_time < index_min_time:
    print("✓ 板块领先大盘触底（先锋板块）")
else:
    print("板块滞后大盘触底")
```

## 应用场景

### 1. 盘面联动分析
- 识别大盘关键变盘节点（急跌低点、V型反转、突破点）
- 计算板块与大盘的时差（领先/滞后）
- 判断板块的主动性（先锋/共振/分离）

### 2. 弹性系数计算
```python
# 计算板块相对大盘的弹性系数
index_change = index_data['change_pct']
sector_change = sector_data['change_pct']

elasticity = sector_change / index_change if index_change != 0 else 0
print(f"弹性系数: {elasticity:.2f}")

if elasticity > 3.0:
    print("✓ 强度共振板块（弹性系数>3）")
```

### 3. 跷跷板效应识别
```python
# 识别某板块拉升时大盘是否跳水
sector_df = sector_data['data']
index_df = index_data['data']

# 找到板块大幅拉升的时间点
sector_surge = sector_df[sector_df['pct_change'] > sector_df['pct_change'].mean() + 1]

for idx in sector_surge.index:
    time = sector_df.loc[idx, 'time']
    # 查找同一时间大盘的表现
    index_at_time = index_df[index_df['time'] == time]
    
    if not index_at_time.empty:
        index_change = index_at_time.iloc[0]['pct_change']
        sector_change = sector_df.loc[idx, 'pct_change']
        
        if sector_change > 1.0 and index_change < -0.5:
            print(f"✓ 检测到跷跷板效应: {time}")
            print(f"  板块涨幅: {sector_change:.2f}%")
            print(f"  大盘跌幅: {index_change:.2f}%")
```

## 与题材复盘智能体的集成

该接口将用于题材复盘智能体的以下功能：

### Requirement 2: 盘面联动与主动性分析
- **2.1**: 识别大盘分时图的关键共振点（急跌低点、V型反转点、突破点）
- **2.2**: 计算每个目标板块与大盘的时差
- **2.3**: 识别先锋板块（领先大盘5-10分钟）
- **2.4**: 识别强度共振板块（弹性系数≥3.0）
- **2.5**: 识别分离板块（大盘跳水时保持平稳）
- **2.6**: 识别跷跷板效应

## 注意事项

1. **时间戳处理**: 分时数据的时间戳已转换为datetime对象，方便时间计算
2. **涨跌幅计算**: pct_change字段是相对昨收价的涨跌幅，已经计算好
3. **数据完整性**: 实时数据可能不完整（交易时间内），历史数据是完整的一天
4. **超时设置**: 默认60秒超时，如果网络较慢可以适当增加
5. **错误处理**: 接口内置了错误处理和日志输出，调用失败会返回空字典

## 后续优化建议
