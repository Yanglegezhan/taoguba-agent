## 🚀 快速使用指南

### 1️⃣ 获取今日完整数据
```python
from kaipanla_crawler import KaipanlaCrawler

crawler = KaipanlaCrawler()
data = crawler.get_daily_data("2026-01-20")

print(f"涨停数: {data['涨停数']}")
print(f"连板率: {data['连板率']}%")
print(f"上证指数: {data['上证指数']}")
```

### 2️⃣ 实时监控连板梯队
```python
# 获取实时全市场连板梯队
realtime = crawler.get_market_limit_up_ladder()

print(f"总涨停: {realtime['statistics']['total_limit_up']}只")
print(f"最高板: {realtime['statistics']['max_consecutive']}连板")
print(f"反包板: {len(realtime['broken_stocks'])}只")

# 查看连板分布
for consecutive, stocks in sorted(realtime['ladder'].items(), reverse=True):
    print(f"{consecutive}连板: {len(stocks)}只")
```

### 3️⃣ 板块连板分析
```python
# 获取板块连板梯队
sectors = crawler.get_sector_limit_up_ladder()

for sector in sectors['sectors'][:10]:  # 前10个板块
    print(f"{sector['sector_name']}: {sector['limit_up_count']}只涨停")
    
    # 查看该板块的连板股票
    for stock in sector['stocks'][:3]:  # 前3只
        print(f"  {stock['stock_name']} {stock['consecutive_days']}连板")
```

### 4️⃣ 涨停原因分析
```python
# 获取涨停原因板块
data = crawler.get_sector_ranking("2026-01-20")

for sector in data['sectors'][:5]:  # 前5个板块
    print(f"\n{sector['sector_name']}: {sector['stock_count']}只")
    for stock in sector['stocks'][:3]:
        print(f"  {stock['股票名称']}")
        print(f"    原因: {stock['涨停原因']}")
        print(f"    连板: {stock['连板描述']}")
```

### 5️⃣ 市场情绪判断

## 📈 典型应用场景

### 场景1: 每日复盘分析
```python
# 获取完整数据
data = crawler.get_daily_data("2026-01-20")

# 市场强弱判断
if data['连板率'] > 60:
    print("市场情绪强势")
elif data['连板率'] > 40:
    print("市场情绪中性")
else:
    print("市场情绪弱势")

# 获取连板梯队详情
ladder = crawler.get_consecutive_limit_up("2026-01-20")
print(f"最高板: {ladder['max_consecutive']}连板")
print(f"龙头: {ladder['max_consecutive_stocks']}")
print(f"题材: {ladder['max_consecutive_concepts']}")
```

### 场景2: 盘中实时监控
```python
import time

while True:
    # 获取实时连板梯队
    realtime = crawler.get_market_limit_up_ladder()
    
    print(f"\n[{realtime['date']}] 实时监控")
    print(f"涨停: {realtime['statistics']['total_limit_up']}只")
    print(f"最高板: {realtime['statistics']['max_consecutive']}连板")
    
    # 计算连板率
    total = realtime['statistics']['total_limit_up']
    first = len(realtime['ladder'].get(1, []))
    rate = ((total - first) / total * 100) if total > 0 else 0
    print(f"连板率: {rate:.1f}%")
    
    # 每5分钟更新一次
    time.sleep(300)
```

### 场景3: 板块轮动分析
```python
# 获取板块连板梯队
sectors = crawler.get_sector_limit_up_ladder()

    sectors['sectors'],
```

## 💡 核心概念

### 连板梯队
- **首板**: 当日首次涨停
- **N连板**: 连续N天涨停
- **反包板**: 涨停→打开→再封板（不计入梯队）
- **打开高度**: 曾达到高度但打开（不计入梯队）

### 连板率
```
连板率 = (连板数 / 总涨停数) × 100%
```
- 连板率 > 60%: 市场强势
- 连板率 40-60%: 市场中性
- 连板率 < 40%: 市场弱势

### 实时 vs 历史
- **实时**: 不传 `date` 参数，盘中实时更新
- **历史**: 传 `date` 参数，收盘后固定数据

## ⚡ 性能提示

1. **批量查询**: 使用 `ThreadPoolExecutor` 并发获取多日数据
2. **数据缓存**: 历史数据可以缓存，避免重复请求
3. **超时设置**: 网络不稳定时增加 `timeout` 参数
4. **实时监控**: 建议5分钟更新一次，避免频繁请求

## 🔧 常见问题

**Q: 如何判断是交易日？**
```python
data = crawler.get_daily_data("2026-01-20")
if data['涨停数'] == 0:
    print("非交易日或无数据")
```

**Q: 如何获取最新交易日数据？**
```python
from datetime import datetime
today = datetime.now().strftime("%Y-%m-%d")
data = crawler.get_daily_data(today)
```

**Q: 反包板和连板有什么区别？**
- 连板: 连续涨停，中间不打开
- 反包板: 涨停后打开再封板，不算连续

**Q: 如何计算市场情绪？**
```python
data = crawler.get_market_limit_up_ladder()
total = data['statistics']['total_limit_up']
first = len(data['ladder'].get(1, []))
rate = ((total - first) / total * 100) if total > 0 else 0

if rate > 60:
    emotion = "强势"
elif rate > 40:
    emotion = "中性"
else:
    emotion = "弱势"
```
