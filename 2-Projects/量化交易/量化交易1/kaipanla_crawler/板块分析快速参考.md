## 🔄 3. 板块轮动分析

```python
# 获取数据
df = crawler.get_sector_strength_ndays("2026-01-20", num_days=7)

# 分割时间段
dates = sorted(df['日期'].unique(), reverse=True)
recent_3days = dates[:3]
previous_4days = dates[3:7]

# 计算强度变化
recent = df[df['日期'].isin(recent_3days)].groupby('板块名称')['涨停数'].sum()
previous = df[df['日期'].isin(previous_4days)].groupby('板块名称')['涨停数'].sum()
change = recent - previous

# 找出强度上升的板块
rising = change.sort_values(ascending=False)
print("强度上升的板块:")
print(rising.head(10))
```

## 💰 4. 资金+强度综合

```python
# 1. 获取强度数据
df = crawler.get_sector_strength_ndays("2026-01-20", num_days=7)
sector_trend = df.groupby('板块名称')['涨停数'].sum().sort_values(ascending=False)

# 2. 获取最强板块的资金数据
top_sector = sector_trend.index[0]
sector_code = df[df['板块名称'] == top_sector].iloc[0]['板块代码']
capital_data = crawler.get_sector_capital_data(sector_code)

# 3. 综合评分
score = (
    sector_trend[top_sector] * 0.4 +                           # 涨停数权重40%
    (capital_data['main_net_inflow'] / 1000000000) * 0.3 +     # 主力净额权重30%
    (capital_data['turnover'] / 100000000000) * 0.3            # 成交额权重30%
)
```

## 🎯 6. 实战示例

### 监控多个板块资金
```python
sectors = {
    "801235": "化工",
    "801346": "电力设备",
    "801225": "机械设备"
}

for code, name in sectors.items():
    data = crawler.get_sector_capital_data(code)
    print(f"{name}: "
          f"成交额 {data['turnover']/100000000:.2f}亿 | "
          f"主力净额 {data['main_net_inflow']/100000000:.2f}亿")
```

### 找出持续强势板块
```python
df = crawler.get_sector_strength_ndays("2026-01-20", num_days=7)

# 计算持续性和总涨停数
persistence = df[df['涨停数'] > 0].groupby('板块名称').size()
total = df.groupby('板块名称')['涨停数'].sum()

# 合并结果
result = pd.DataFrame({
    '持续天数': persistence,
    '总涨停数': total
}).sort_values('总涨停数', ascending=False)

print(result.head(10))
```

### 对比板块强度变化
```python
df = crawler.get_sector_strength_ndays("2026-01-20", num_days=7)

# 选择前3个最强板块
top_sectors = df.groupby('板块名称')['涨停数'].sum().nlargest(3).index

# 创建透视表
pivot = df[df['板块名称'].isin(top_sectors)].pivot_table(
    index='日期',
    columns='板块名称',
    values='涨停数',
    fill_value=0
)

print(pivot)
```

## ⚡ 7. 性能优化

### 批量获取资金数据
```python
from concurrent.futures import ThreadPoolExecutor

sectors = ["801235", "801346", "801225"]

with ThreadPoolExecutor(max_workers=3) as executor:
    results = list(executor.map(crawler.get_sector_capital_data, sectors))
```

### 缓存历史数据
```python
# 缓存字典
cache = {}

def get_cached_strength(date, num_days):
    key = f"{date}_{num_days}"
    if key not in cache:
        cache[key] = crawler.get_sector_strength_ndays(date, num_days)
    return cache[key]
```

## 💡 提示

1. **实时数据**: 不传date参数获取实时数据
2. **历史数据**: 传入date参数获取历史数据
3. **交易日**: 非交易日会自动跳过
4. **超时**: 默认60秒，可以调整timeout参数
5. **单位**: 注意成交额和市值的单位转换
