# 实时连板数据功能说明

## 更新日期
2026-01-20

## 新增功能概述

新增4个实时连板数据获取函数，用于获取盘中实时的连板梯队、涨跌停数据和各连板股票列表。

---

## 1. 获取实时连板梯队指数

### 函数签名
```python
def get_realtime_limit_up_index(self, timeout=60):
```

### 功能说明
获取实时连板梯队指数，包括首板、二板、三板、四板以上的家数。

### API接口
```
POST https://apphwhq.longhuvip.com/w1/api/index.php
参数: a=DailyLimitIndex&c=HomeDingPan
```

### 返回字段
| 字段 | 类型 | 说明 |
|------|------|------|
| first_board | int | 首板家数 |
| second_board | int | 二板家数 |
| third_board | int | 三板家数 |
| fourth_board_plus | int | 四板以上家数 |
| total_limit_up | int | 总涨停数 |
| consecutive_rate | float | 连板率（%） |

### 使用示例
```python
from kaipanla_crawler import KaipanlaCrawler

crawler = KaipanlaCrawler()

# 获取实时连板梯队指数
data = crawler.get_realtime_limit_up_index()

print(f"首板: {data['first_board']}只")
print(f"二板: {data['second_board']}只")
print(f"三板: {data['third_board']}只")
print(f"四板以上: {data['fourth_board_plus']}只")
print(f"总涨停数: {data['total_limit_up']}只")
print(f"连板率: {data['consecutive_rate']:.2f}%")
```

## 2. 获取实时实际涨跌停数据

### 函数签名
```python
def get_realtime_actual_limit_up_down(self, timeout=60):
```

### 功能说明
获取实时实际涨跌停数据，区分实际涨跌停和包含一字板的涨跌停。

### API接口
```
POST https://apphwhq.longhuvip.com/w1/api/index.php
参数: a=MarketStockZDNum&c=HomeDingPan
```

### 返回字段
| 字段 | 类型 | 说明 |
|------|------|------|
| actual_limit_up | int | 实际涨停数（不含一字板） |
| actual_limit_down | int | 实际跌停数（不含一字板） |
| limit_up | int | 涨停数（含一字板） |
| limit_down | int | 跌停数（含一字板） |

### 使用示例
```python
from kaipanla_crawler import KaipanlaCrawler

crawler = KaipanlaCrawler()

# 获取实时实际涨跌停数据
data = crawler.get_realtime_actual_limit_up_down()

print(f"实际涨停: {data['actual_limit_up']}只")
print(f"实际跌停: {data['actual_limit_down']}只")
print(f"涨停（含一字板）: {data['limit_up']}只")
print(f"跌停（含一字板）: {data['limit_down']}只")
```

## 3. 获取实时指定连板股票

### 函数签名
```python
def get_realtime_board_stocks(self, board_type=1, timeout=60):
```

### 功能说明
获取实时指定连板类型的股票列表。

### API接口
```
POST https://apphwhq.longhuvip.com/w1/api/index.php
参数: a=DailyLimitPerformance&c=HomeDingPan&PidType={board_type}
```

### 参数说明
| 参数 | 类型 | 说明 |
|------|------|------|
| board_type | int | 连板类型：1=首板, 2=二板, 3=三板, 4=四板, 5=五板及以上 |
| timeout | int | 超时时间（秒），默认60秒 |

### 返回字段
返回股票列表，每个股票包含：
- stock_code: 股票代码
- stock_name: 股票名称
- board_type: 连板类型
- raw_data: 原始数据

### 使用示例
```python
from kaipanla_crawler import KaipanlaCrawler

crawler = KaipanlaCrawler()

# 获取首板股票
first_board = crawler.get_realtime_board_stocks(board_type=1)
print(f"首板股票: {len(first_board)}只")
for stock in first_board[:5]:
    print(f"  {stock['stock_name']} ({stock['stock_code']})")

# 获取二板股票
second_board = crawler.get_realtime_board_stocks(board_type=2)
print(f"\n二板股票: {len(second_board)}只")
for stock in second_board[:5]:
    print(f"  {stock['stock_name']} ({stock['stock_code']})")

# 获取五板及以上股票（龙头）
leaders = crawler.get_realtime_board_stocks(board_type=5)
print(f"\n五板及以上股票（龙头）: {len(leaders)}只")
for stock in leaders:
    print(f"  {stock['stock_name']} ({stock['stock_code']})")
```

## 4. 获取实时所有连板股票

### 函数签名
```python
def get_realtime_all_boards_stocks(self, timeout=60):
```

### 功能说明
一次性获取实时所有连板的股票列表（首板到五板以上）。

### 返回字段
| 字段 | 类型 | 说明 |
|------|------|------|
| first_board | list | 首板股票列表 |
| second_board | list | 二板股票列表 |
| third_board | list | 三板股票列表 |
| fourth_board | list | 四板股票列表 |
| fifth_board_plus | list | 五板及以上股票列表 |
| statistics | dict | 统计信息 |

### statistics字段
| 字段 | 类型 | 说明 |
|------|------|------|
| total_stocks | int | 总涨停股票数 |
| first_board_count | int | 首板数量 |
| second_board_count | int | 二板数量 |
| third_board_count | int | 三板数量 |
| fourth_board_count | int | 四板数量 |
| fifth_board_plus_count | int | 五板及以上数量 |
| consecutive_rate | float | 连板率（%） |

### 使用示例
```python
from kaipanla_crawler import KaipanlaCrawler

crawler = KaipanlaCrawler()

# 获取所有连板股票
data = crawler.get_realtime_all_boards_stocks()

# 查看统计信息
stats = data['statistics']
print(f"首板: {stats['first_board_count']}只")
print(f"二板: {stats['second_board_count']}只")
print(f"三板: {stats['third_board_count']}只")
print(f"四板: {stats['fourth_board_count']}只")
print(f"五板以上: {stats['fifth_board_plus_count']}只")
print(f"总计: {stats['total_stocks']}只")
print(f"连板率: {stats['consecutive_rate']:.2f}%")

# 查看五板以上的股票（龙头）
print(f"\n五板以上股票（龙头）:")
for stock in data['fifth_board_plus']:
    print(f"  {stock['stock_name']} ({stock['stock_code']})")
```

## 5. 板块内实时涨停原因（已有功能）

### 函数签名
```python
def get_sector_ranking(self, date=None, index=0, timeout=1600):
```

### 功能说明
获取板块内的涨停股票及其涨停原因。**此函数已存在**，使用实时API，可以获取板块内每只股票的涨停原因。

### 返回字段
每个板块包含：
- sector_code: 板块代码
- sector_name: 板块名称
- stock_count: 涨停股票数量
- stocks: 涨停股票列表
  - 股票代码
  - 股票名称
  - **涨停原因** ⭐
  - 连板天数
  - 涨停时间
  - 成交额
  - 主力资金
  - 等等

### 使用示例
```python
from kaipanla_crawler import KaipanlaCrawler

crawler = KaipanlaCrawler()

# 获取板块涨停原因数据
data = crawler.get_sector_ranking()

# 遍历板块
for sector in data['sectors'][:5]:  # 前5个板块
    print(f"\n{sector['sector_name']}: {sector['stock_count']}只涨停")
    
    # 查看该板块的涨停股票及原因
    for stock in sector['stocks'][:3]:  # 前3只
        print(f"  {stock['股票名称']} ({stock['股票代码']})")
        print(f"    涨停原因: {stock['涨停原因']}")
        print(f"    连板天数: {stock['连板天数']}")
        print(f"    涨停时间: {stock['涨停时间']}")
```

**输出示例**:
```
化工: 13只涨停
  新华百货 (600785)
    涨停原因: 政策利好
    连板天数: 4
    涨停时间: 09:30:00
  ...
```

## 应用场景

### 场景1: 实时监控连板梯队
```python
import time

crawler = KaipanlaCrawler()

while True:
    # 获取实时连板梯队指数
    data = crawler.get_realtime_limit_up_index()
    
    print(f"\n[{datetime.now().strftime('%H:%M:%S')}] 实时连板梯队")
    print(f"首板: {data['first_board']}只")
    print(f"二板: {data['second_board']}只")
    print(f"三板: {data['third_board']}只")
    print(f"四板以上: {data['fourth_board_plus']}只")
    print(f"连板率: {data['consecutive_rate']:.2f}%")
    
    # 每1分钟更新一次
    time.sleep(60)
```

### 场景2: 龙头股追踪
```python
crawler = KaipanlaCrawler()

# 获取所有连板股票
data = crawler.get_realtime_all_boards_stocks()

# 找出龙头股（五板及以上）
leaders = data['fifth_board_plus']

if leaders:
    print("市场龙头股:")
    for stock in leaders:
        print(f"  {stock['stock_name']} ({stock['stock_code']})")
else:
    print("当前无五板及以上龙头股")
```

### 场景3: 连板率监控
```python
crawler = KaipanlaCrawler()

# 获取连板梯队指数
data = crawler.get_realtime_limit_up_index()

# 判断市场情绪
consecutive_rate = data['consecutive_rate']

if consecutive_rate > 60:
    print("市场情绪: 强势（连板率>60%）")
elif consecutive_rate > 40:
    print("市场情绪: 中性（连板率40-60%）")
else:
    print("市场情绪: 弱势（连板率<40%）")
```

### 场景4: 板块涨停原因分析
```python
crawler = KaipanlaCrawler()

# 获取板块涨停原因
data = crawler.get_sector_ranking()

# 统计涨停原因
reasons = {}
for sector in data['sectors']:
    for stock in sector['stocks']:
        reason = stock['涨停原因']
        reasons[reason] = reasons.get(reason, 0) + 1

# 显示最常见的涨停原因
print("今日最常见的涨停原因:")
for reason, count in sorted(reasons.items(), key=lambda x: x[1], reverse=True)[:5]:
    print(f"  {reason}: {count}只")
```

## 注意事项

1. **交易时间**: 这些函数只在交易时间内返回有效数据
2. **实时性**: 数据实时更新，建议每1-5分钟刷新一次
3. **API限制**: 避免频繁请求，建议设置合理的刷新间隔
4. **数据延迟**: 实时数据可能有几秒到几十秒的延迟
5. **非交易时间**: 非交易时间返回的数据为0或空

## 完整示例

```python
from kaipanla_crawler import KaipanlaCrawler
from datetime import datetime
import time

crawler = KaipanlaCrawler()

def monitor_market():
    """实时监控市场情绪"""
    while True:
        print("\n" + "=" * 60)
        print(f"[{datetime.now().strftime('%Y-%m-%d %H:%M:%S')}] 市场实时监控")
        print("=" * 60)
        
        # 1. 获取连板梯队指数
        ladder_data = crawler.get_realtime_limit_up_index()
        print(f"\n【连板梯队】")
        print(f"首板: {ladder_data['first_board']}只")
        print(f"二板: {ladder_data['second_board']}只")
        print(f"三板: {ladder_data['third_board']}只")
        print(f"四板以上: {ladder_data['fourth_board_plus']}只")
        print(f"连板率: {ladder_data['consecutive_rate']:.2f}%")
        
        # 2. 获取实际涨跌停
        limit_data = crawler.get_realtime_actual_limit_up_down()
        print(f"\n【涨跌停】")
        print(f"实际涨停: {limit_data['actual_limit_up']}只")
        print(f"实际跌停: {limit_data['actual_limit_down']}只")
        
        # 3. 获取龙头股
        all_boards = crawler.get_realtime_all_boards_stocks()
        leaders = all_boards['fifth_board_plus']
        if leaders:
            print(f"\n【龙头股】（五板及以上）")
            for stock in leaders:
                print(f"  {stock['stock_name']} ({stock['stock_code']})")
        
        # 4. 判断市场情绪
        print(f"\n【市场情绪】")
        rate = ladder_data['consecutive_rate']
        if rate > 60:
            print("强势（连板率>60%）")
        elif rate > 40:
            print("中性（连板率40-60%）")
        else:
            print("弱势（连板率<40%）")
        
        # 每5分钟更新一次
        time.sleep(300)

if __name__ == "__main__":
    monitor_market()
```
