## 常用字段快速索引

### 基本信息
- `[0]` 股票代码
- `[1]` 股票名称
- `[4]` 所属板块

### 价格相关
- `[5]` 价格
- `[6]` 涨幅（%）
- `[56]` 价格（重复）
- `[57]` 涨幅（重复）

### 成交相关
- `[7]` 成交额
- `[8]` 实际换手率（%）
- `[25]` 换手率（%）
- `[21]` 量比

### 竞价相关
- `[34]` 竞价量比
- `[35]` 竞价金额
- `[36]` 竞价涨幅（%）

### 资金流向
- `[11]` 主力买
- `[12]` 主力卖
- `[13]` 主力净额
- `[50]` 300w以上大单净额

### 市值相关
- `[10]` 实际流通市值
- `[37]` 总市值
- `[38]` 流通市值

### 连板信息
- `[23]` 连板天数（如"10天7板"、"首板"、"2连板"）
- `[24]` 板块地位（如"龙一"、"龙二"）
- `[28]` 收盘封单
- `[29]` 最大封单

### 估值指标
- `[53]` 市净率（PB）
- `[60]` PE（动）
- `[61]` PE TTM
- `[62]` PE（静）

### 其他
- `[33]` 振幅（%）
- `[40]` 领涨次数
- `[58]` 人气值
- `[59]` 排名变化

## 使用示例

### 示例1：提取关键字段

```python
from kaipanla_crawler import KaipanlaCrawler

crawler = KaipanlaCrawler()
data = crawler.get_sector_constituent_stocks("801235", "2026-02-10")

for stock in data['stocks']:
    # 基本信息
    code = stock[0]           # 股票代码
    name = stock[1]           # 股票名称
    sectors = stock[4]        # 所属板块
    
    # 价格信息
    price = stock[5]          # 价格
    change_pct = stock[6]     # 涨幅
    
    # 成交信息
    turnover = stock[7]       # 成交额
    turnover_rate = stock[8]  # 实际换手率
    volume_ratio = stock[21]  # 量比
    
    # 竞价信息
    bid_volume_ratio = stock[34]  # 竞价量比
    bid_amount = stock[35]        # 竞价金额
    bid_change = stock[36]        # 竞价涨幅
    
    # 资金流向
    main_buy = stock[11]      # 主力买
    main_sell = stock[12]     # 主力卖
    main_net = stock[13]      # 主力净额
    
    # 连板信息
    board_desc = stock[23]    # 连板描述
    leader_mark = stock[24]   # 龙头标记
    
    # 市值
    total_mv = stock[37]      # 总市值
    circulating_mv = stock[38] # 流通市值
    
    # 估值
    pb = stock[53]            # 市净率
    pe_dynamic = stock[60]    # PE动
    pe_ttm = stock[61]        # PETTM
    pe_static = stock[62]     # PE静
    
    print(f"{code} {name}: {price} ({change_pct:+.2f}%) 竞价涨幅{bid_change:.2f}%")
```

### 示例2：筛选龙头股

```python
from kaipanla_crawler import KaipanlaCrawler

        leaders.append({
            'code': stock[0],
            'name': stock[1],
            'price': stock[5],
            'change': stock[6],
            'board': stock[23],
            'leader': stock[24],
            'main_net': stock[13]
        })

print(f"板块龙头股: {len(leaders)} 只")
for s in leaders:
    print(f"{s['leader']} {s['code']} {s['name']}: {s['change']:.2f}% {s['board']}")
    print(f"  主力净额: {s['main_net']/100000000:.2f}亿")
```

### 示例4：连板股票统计

```python
from kaipanla_crawler import KaipanlaCrawler

crawler = KaipanlaCrawler()
data = crawler.get_sector_constituent_stocks("801235", "2026-02-10")

# 统计连板情况
board_stats = {}
for stock in data['stocks']:
    board_desc = stock[23]
    if board_desc:
        board_stats[board_desc] = board_stats.get(board_desc, 0) + 1

print("连板统计:")
for board, count in sorted(board_stats.items(), reverse=True):
    print(f"  {board}: {count}只")
```

### 示例5：竞价异动分析

```python
from kaipanla_crawler import KaipanlaCrawler

crawler = KaipanlaCrawler()
data = crawler.get_sector_constituent_stocks("801235", "2026-02-10")

# 筛选竞价异动股票（竞价量比>1 且 竞价涨幅>5%）
bid_anomaly = []
for stock in data['stocks']:
    bid_volume_ratio = stock[34]  # 竞价量比
    bid_change = stock[36]        # 竞价涨幅
    
    if bid_volume_ratio > 1 and bid_change > 5:
        bid_anomaly.append({
            'code': stock[0],
            'name': stock[1],
            'bid_volume_ratio': bid_volume_ratio,
            'bid_amount': stock[35],
            'bid_change': bid_change,
            'final_change': stock[6]  # 最终涨幅
        })

# 按竞价涨幅排序
bid_anomaly.sort(key=lambda x: x['bid_change'], reverse=True)

print(f"竞价异动股票: {len(bid_anomaly)} 只")
for s in bid_anomaly:
    print(f"{s['name']}: 竞价涨幅{s['bid_change']:.2f}% -> 最终涨幅{s['final_change']:.2f}%")
    print(f"  竞价量比: {s['bid_volume_ratio']:.2f}, 竞价金额: {s['bid_amount']/100000000:.2f}亿")
```

### 示例6：竞价与收盘对比

```python
from kaipanla_crawler import KaipanlaCrawler

crawler = KaipanlaCrawler()
data = crawler.get_sector_constituent_stocks("801235", "2026-02-10")

# 对比竞价涨幅和最终涨幅
print("竞价涨幅 vs 最终涨幅对比:")
print(f"{'股票':<12} {'竞价涨幅':<10} {'最终涨幅':<10} {'差异':<10} {'兑现率':<10}")
print("-" * 60)

for stock in data['stocks'][:10]:  # 前10只
    name = stock[1]
    bid_change = stock[36]      # 竞价涨幅
    final_change = stock[6]     # 最终涨幅
    diff = final_change - bid_change
    realize_rate = (final_change / bid_change * 100) if bid_change != 0 else 0
    
    print(f"{name:<12} {bid_change:>9.2f}% {final_change:>9.2f}% {diff:>9.2f}% {realize_rate:>9.1f}%")
```

## 注意事项

1. **重复字段**：
   - 价格字段：索引5和56都是价格
   - 涨幅字段：索引6、54、57都是涨幅
   - 建议统一使用索引5和6

2. **空字段**：
   - 索引2、26、30、31、32、39 通常为空字符串
   - 索引3、9、22、27、41、43、46、51、55 通常为0

3. **连板描述格式**：
   - "首板"：当日首次涨停
   - "2连板"：连续2天涨停
   - "10天7板"：10天内7次涨停（不连续）

4. **龙头标记**：
   - "龙一"、"龙二"、"龙三"：板块内的龙头地位
   - 空字符串：非龙头股

5. **主力资金**：
   - 主力买（索引11）：正数
   - 主力卖（索引12）：负数
   - 主力净额（索引13）= 主力买 + 主力卖

6. **竞价数据**：
   - 竞价量比（索引34）：竞价阶段的量比
   - 竞价金额（索引35）：竞价阶段的成交金额
   - 竞价涨幅（索引36）：竞价阶段的涨跌幅
   - 可用于盘前分析和预判开盘走势

7. **竞价与收盘对比**：
   - 竞价涨幅高但最终涨幅低：可能是高开低走
   - 竞价涨幅低但最终涨幅高：可能是低开高走
   - 兑现率 = 最终涨幅 / 竞价涨幅 × 100%
