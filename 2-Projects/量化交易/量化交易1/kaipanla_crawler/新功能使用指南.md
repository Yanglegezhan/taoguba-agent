# 新功能使用指南

本文档介绍 kaipanla_crawler v1.1.0 新增的7个功能的使用方法。

## 快速开始

```python
from kaipanla_crawler import KaipanlaCrawler

# 创建爬虫实例
crawler = KaipanlaCrawler()
```

## 功能列表

### 1. 获取板块资金数据 `get_sector_capital_data()`

获取指定板块的资金成交额、主力净额、涨跌幅等14个关键指标。

**参数：**
- `sector_code` (str): 板块代码，如 "801235"（计算机设备）
- `date` (str, 可选): 日期，格式 "YYYY-MM-DD"。不传则获取实时数据
- `timeout` (int): 超时时间，默认60秒

**返回：** dict，包含14个字段

**示例：**
```python
# 获取实时数据
data = crawler.get_sector_capital_data("801235")
print(f"成交额: {data['成交额'] / 1e8:.2f}亿")
print(f"主力净额: {data['主力净额'] / 1e8:.2f}亿")

# 获取历史数据
data = crawler.get_sector_capital_data("801235", date="2026-01-19")
```

**返回字段：**
- 成交额、涨跌幅、换手率、主力净额、主力净额占比、主力净额占成交额比
- 上涨家数、下跌家数、涨停家数、流通市值、总市值、市盈率等

---

### 2. 获取N日板块强度排名 `get_sector_strength_ndays()`

获取最近N个交易日的板块强度排名（基于涨停数）。

**参数：**
- `end_date` (str): 结束日期，格式 "YYYY-MM-DD"
- `num_days` (int): 天数，默认7天
- `timeout` (int): 超时时间，默认60秒

**返回：** pandas.DataFrame

**示例：**
```python
# 获取最近7天的板块强度
df = crawler.get_sector_strength_ndays("2026-01-20", num_days=7)

# 查看数据
print(df.head())

# 统计每个板块的总涨停数
sector_stats = df.groupby('板块名称')['涨停数'].sum().sort_values(ascending=False)
print(sector_stats.head(10))
```

---

### 3. 获取实时连板梯队指数 `get_realtime_limit_up_index()`

获取实时的首板、二板、三板、四板以上的家数统计。

**参数：**
- `timeout` (int): 超时时间，默认60秒

**返回：** dict

**示例：**
```python
index_data = crawler.get_realtime_limit_up_index()

print(f"首板: {index_data['首板家数']}家")
print(f"二板: {index_data['二板家数']}家")
print(f"三板: {index_data['三板家数']}家")
print(f"四板以上: {index_data['四板以上家数']}家")
```

---

### 4. 获取实时实际涨跌停数据 `get_realtime_actual_limit_up_down()`

获取实时的实际涨停、跌停家数。

**参数：**
- `timeout` (int): 超时时间，默认60秒

**返回：** dict

**示例：**
```python
limit_data = crawler.get_realtime_actual_limit_up_down()

print(f"实际涨停: {limit_data['实际涨停']}家")
print(f"实际跌停: {limit_data['实际跌停']}家")
```

---

### 5. 获取指定连板股票列表 `get_realtime_board_stocks()`

获取指定连板（首板/二板/三板等）的详细股票列表，包含23个字段。

**参数：**
- `board_type` (int): 连板类型，1=首板, 2=二板, 3=三板, 4=四板, 5=五板以上
- `timeout` (int): 超时时间，默认60秒

**返回：** list[dict]，每个股票包含23个字段

**示例：**
```python
# 获取首板股票
first_board = crawler.get_realtime_board_stocks(board_type=1)

for stock in first_board[:5]:
    print(f"{stock['股票名称']} ({stock['股票代码']})")
    print(f"  涨停原因: {stock['涨停原因']}")
    print(f"  成交额: {stock['成交额'] / 1e8:.2f}亿")
    print(f"  主力净额: {stock['主力净额'] / 1e8:.2f}亿")
    print(f"  封单额: {stock['封单额'] / 1e8:.2f}亿")
```

**返回字段：**
- 股票代码、股票名称、连板类型、时间戳、涨停原因
- 成交额、流通市值、主力买入、主力卖出、主力净额
- 封单额、概念标签、总市值、振幅、连板天数
- 涨跌幅、板块代码、板块涨停数、涨停价、涨停幅度等

---

### 6. 获取所有连板股票 `get_realtime_all_boards_stocks()`

一次性获取所有连板（首板到五板以上）的股票列表。

**参数：**
- `timeout` (int): 超时时间，默认60秒

**返回：** dict，包含各连板的股票列表和统计信息

**示例：**
```python
all_boards = crawler.get_realtime_all_boards_stocks()

# 查看各连板家数
for board_name, stocks in all_boards.items():
    if board_name != 'statistics':
        print(f"{board_name}: {len(stocks)}只")

# 查看统计信息
stats = all_boards['statistics']
print(f"\n总计: {stats['total_count']}只涨停股票")
```

---

### 7. 简洁接口：获取连板数量和列表 `get_board_stocks_count_and_list()`

**推荐使用**：这是最简洁的接口，返回指定连板的数量和股票列表。

**参数：**
- `board_type` (int): 连板类型，1=首板, 2=二板, 3=三板, 4=四板, 5=五板以上
- `timeout` (int): 超时时间，默认60秒

**返回：** tuple (count, stocks_list)
- `count` (int): 股票数量
- `stocks_list` (list[dict]): 股票列表

**示例：**
```python
# 获取首板数量和列表
count, stocks = crawler.get_board_stocks_count_and_list(1)

print(f"首板共 {count} 只股票")

# 遍历股票
for stock in stocks:
    print(f"{stock['股票名称']}: {stock['涨停原因']}")
```

## 完整示例：市场情绪分析

```python
from kaipanla_crawler import KaipanlaCrawler

def analyze_market_sentiment():
    """综合分析市场情绪"""
    crawler = KaipanlaCrawler()
    
    print("=" * 60)
    print("市场情绪分析")
    print("=" * 60)
    
    # 1. 连板梯队
    print("\n【连板梯队】")
    index_data = crawler.get_realtime_limit_up_index()
    print(f"首板: {index_data['首板家数']}家")
    print(f"二板: {index_data['二板家数']}家")
    print(f"三板: {index_data['三板家数']}家")
    print(f"四板以上: {index_data['四板以上家数']}家")
    
    # 2. 涨跌停统计
    print("\n【涨跌停统计】")
    limit_data = crawler.get_realtime_actual_limit_up_down()
    print(f"实际涨停: {limit_data['实际涨停']}家")
    print(f"实际跌停: {limit_data['实际跌停']}家")
    
    # 3. 首板分析
    print("\n【首板TOP5】")
    count, stocks = crawler.get_board_stocks_count_and_list(1)
    print(f"首板共 {count} 只")
    
    # 按主力净额排序
    sorted_stocks = sorted(stocks, key=lambda x: x['主力净额'], reverse=True)
    for i, stock in enumerate(sorted_stocks[:5], 1):
        print(f"{i}. {stock['股票名称']} - {stock['涨停原因']}")
        print(f"   主力净额: {stock['主力净额'] / 1e8:.2f}亿")
    
    # 4. 热门板块
    print("\n【热门板块（7日强度）】")
    df = crawler.get_sector_strength_ndays("2026-01-20", num_days=7)
    sector_stats = df.groupby('板块名称')['涨停数'].sum().sort_values(ascending=False)
    for sector, count in sector_stats.head(5).items():
        print(f"{sector}: {count}只涨停")
    
    print("\n" + "=" * 60)

if __name__ == "__main__":
    analyze_market_sentiment()
```

## 注意事项

1. **实时 vs 历史数据**：
   - 实时数据使用 `apphwhq.longhuvip.com`
   - 历史数据使用 `apphis.longhuvip.com`
   - `get_sector_capital_data()` 会自动根据是否传入 `date` 参数选择API

2. **非交易时间**：
   - 非交易时间调用实时接口，数据可能为0或空
   - 建议在交易时间（9:30-15:00）使用实时接口

3. **连板类型**：
   - 1 = 首板
   - 2 = 二板
   - 3 = 三板
   - 4 = 四板
   - 5 = 五板及以上

4. **超时设置**：
   - 默认超时60秒
   - 网络不稳定时可适当增加
