# 板块竞价异动功能说明

## 功能概述

获取板块竞价异动数据，用于盘前分析和捕捉市场热点。

## 接口说明

### get_sector_bidding_anomaly() - 获取板块竞价异动

**功能**：获取板块竞价异动数据，支持实时和历史数据

**特点**：
- 支持实时数据（盘前竞价时段）
- 支持历史数据查询
- 分类展示：今日新增、昨日延续、其他异动
- 包含竞价爆量、异动金额、板块强度等关键指标

**使用场景**：
- 盘前分析市场热点
- 捕捉板块异动机会
- 跟踪板块延续性
- 资金流向分析

---

## 数据结构说明

### 返回数据结构

```python
{
    "date": "2026-02-11",           # 日期
    "is_realtime": True,             # 是否为实时数据
    "list1": [...],                  # 今日新增竞价异动板块
    "list2": [...],                  # 昨日爆发板块延续异动
    "list3": [...],                  # 其他异动板块
    "total_count": 15                # 总异动板块数量
}
```

### 板块数据字段

每个板块是一个包含6个元素的列表：

| 索引 | 字段名 | 数据类型 | 说明 | 示例 |
|------|--------|----------|------|------|
| 0 | 板块代码 | str | 板块ID | "801005" |
| 1 | 板块名称 | str | 板块中文名称 | "新能源汽车" |
| 2 | 竞价爆量 | float | 竞价量比 | 4.1 |
| 3 | 异动金额 | int | 异动金额（元） | 292197185 |
| 4 | 竞价板块强度 | int | 板块强度指标 | -310 |
| 5 | 主力净额 | int | 主力净流入（元） | -69828031 |

### 三个列表的含义

1. **List1 - 今日新增竞价异动**
   - 今日首次出现竞价异动的板块
   - 通常是新热点或突发事件驱动
   - 关注度最高

2. **List2 - 昨日爆发板块延续异动**
   - 昨日已经爆发，今日继续异动的板块
   - 表示热点延续性强
   - 可能是持续性机会

3. **List3 - 其他异动板块**
   - 其他类型的异动板块
   - 可能是弱异动或特殊情况

```python
    print(f"  板块强度: {strength}")
    print(f"  主力净额: {main_net/100000000:.2f}亿")
```

### 示例3：筛选强势板块

```python
from kaipanla_crawler import KaipanlaCrawler

crawler = KaipanlaCrawler()

# 获取实时竞价异动
data = crawler.get_sector_bidding_anomaly()

# 筛选强势板块（竞价爆量>5 且 板块强度>0）
strong_sectors = []
for sector in data['list1'] + data['list2'] + data['list3']:
    volume_ratio = sector[2]
    strength = sector[4]
    
    if volume_ratio > 5 and strength > 0:
        strong_sectors.append({
            'code': sector[0],
            'name': sector[1],
            'volume_ratio': volume_ratio,
            'amount': sector[3],
            'strength': strength,
            'main_net': sector[5]
        })

# 按板块强度排序
strong_sectors.sort(key=lambda x: x['strength'], reverse=True)

print(f"强势板块: {len(strong_sectors)} 个")
for s in strong_sectors:
    print(f"{s['name']}: 爆量{s['volume_ratio']} 强度{s['strength']}")
```

## 指标说明

### 竞价爆量

- **定义**：竞价阶段的量比
- **计算**：竞价成交量 / 平均成交量
- **意义**：
  - > 5：强烈异动
  - 3-5：明显异动
  - < 3：一般异动

### 板块强度

- **定义**：板块整体强度指标
- **意义**：
  - 正数：板块强势
  - 负数：板块弱势
  - 绝对值越大，强度越明显

### 主力净额

- **定义**：主力资金净流入金额
- **意义**：
  - 正数：主力净流入
  - 负数：主力净流出
  - 配合竞价爆量判断资金动向

## 使用技巧

### 1. 盘前分析流程

```python
# 1. 获取竞价异动
data = crawler.get_sector_bidding_anomaly()

# 2. 筛选强势板块（爆量>5 且 强度>0）
strong = [s for s in data['list1'] if s[2] > 5 and s[4] > 0]

# 3. 查看昨日延续板块
continued = data['list2']

# 4. 结合资金流向
main_inflow = [s for s in strong if s[5] > 0]

# 5. 确定重点关注板块
focus_sectors = main_inflow[:3]
```

### 2. 热点延续性判断

```python
# 获取今日和昨日数据
today = crawler.get_sector_bidding_anomaly()
yesterday = crawler.get_sector_bidding_anomaly("2026-02-10")

# 提取昨日新增板块代码
yesterday_new = set([s[0] for s in yesterday['list1']])

# 检查今日延续情况
continued_codes = set([s[0] for s in today['list2']])

# 计算延续率
if yesterday_new:
    continue_rate = len(yesterday_new & continued_codes) / len(yesterday_new)
    print(f"热点延续率: {continue_rate*100:.1f}%")
```

### 3. 板块轮动分析

```python
# 获取最近3天数据
dates = ["2026-02-11", "2026-02-10", "2026-02-09"]
sector_history = {}

for date in dates:
    data = crawler.get_sector_bidding_anomaly(date)
    for sector in data['list1']:
        code = sector[0]
        name = sector[1]
        if code not in sector_history:
            sector_history[code] = {'name': name, 'dates': []}
        sector_history[code]['dates'].append(date)

# 找出连续异动的板块
continuous = {k: v for k, v in sector_history.items() if len(v['dates']) >= 2}

print("连续异动板块:")
for code, info in continuous.items():
    print(f"{info['name']}: {len(info['dates'])}天")
```

## 注意事项

1. **时间窗口**：
   - 竞价时段：9:15-9:25
   - 建议在9:25后获取实时数据
   - 历史数据在收盘后固定

2. **数据解读**：
   - List1（今日新增）：关注度最高，可能是新热点
   - List2（昨日延续）：热点延续性强，可能有持续性
   - List3（其他异动）：参考价值相对较低

3. **指标配合**：
   - 竞价爆量 + 板块强度：判断异动强度
   - 主力净额：判断资金方向
   - 昨日延续：判断热点持续性

4. **风险提示**：
   - 竞价异动不代表全天走势
   - 需结合其他指标综合判断
   - 注意假突破和诱多

## 常见问题

### Q1: 什么时候使用实时数据？

A: 盘前（9:15-9:30）使用实时数据，用于当日开盘前的分析和决策。

### Q2: 竞价爆量多少算强？

A: 
- > 5：强烈异动，重点关注
- 3-5：明显异动，可以关注
- < 3：一般异动，参考价值有限

### Q3: 板块强度为负数是什么意思？

A: 板块强度为负数表示板块整体偏弱，即使有竞价异动，也可能是弱势反弹或诱多。

### Q4: 如何判断热点是否延续？

A: 查看 List2（昨日延续）的数量和质量：
- List2 数量多：热点延续性强
- List2 数量少或为空：热点切换

### Q5: 主力净额和竞价爆量矛盾怎么办？

A: 
- 爆量但主力净流出：可能是散户追高，需谨慎
- 爆量且主力净流入：资金共振，相对安全
