# 龙虎榜功能说明

## 功能概述

kaipanla_crawler 已经包含完整的龙虎榜数据获取功能，支持获取上榜个股列表和个股详细数据。

## 可用接口

### 1. get_longhubang_stock_list() - 获取龙虎榜上榜个股列表

获取当日或历史的龙虎榜上榜个股列表。

**参数：**
- `date`: 日期，格式YYYY-MM-DD，默认为None（获取当日数据）
- `index`: 分页索引，默认0（第一页）
- `page_size`: 每页数量，默认500条
- `timeout`: 超时时间（秒），默认1600秒

**返回数据结构：**
```python
{
    "date": "2026-02-11",           # 日期
    "user_type": 0,                 # 用户类型
    "stocks": [                     # 股票列表
        {
            "stock_code": "002342",         # 股票代码
            "stock_name": "巨力索具",        # 股票名称
            "change_pct": "10.04",          # 涨跌幅
            "reason_type": "日涨幅偏离值达7%", # 上榜原因类型
            "buy_amount": 123456789.0,      # 买入金额
            "join_num": 5,                  # 上榜营业部数量
            "turnover": 500000000,          # 成交额
            "circulating_market_cap": 3000000000,  # 流通市值
            "amplitude": "12.5",            # 振幅
            "turnover_ratio": "15.6",       # 换手率
            "total_market_cap": 5000000000  # 总市值
        }
    ],
    "total_count": 50               # 总数量
}
```

**使用示例：**
```python
from kaipanla_crawler import KaipanlaCrawler

crawler = KaipanlaCrawler()

# 获取当日龙虎榜
result = crawler.get_longhubang_stock_list()
print(f"今日上榜个股数量: {result['total_count']}")

# 获取历史龙虎榜
result = crawler.get_longhubang_stock_list("2026-02-10")

# 遍历上榜个股
for stock in result['stocks']:
    print(f"{stock['stock_name']} ({stock['stock_code']}): {stock['change_pct']}%")
    print(f"  上榜原因: {stock['reason_type']}")
    print(f"  买入金额: {stock['buy_amount']:,.0f}")
```

### 2. get_longhubang_stock_detail() - 获取个股龙虎榜详细数据

获取指定个股的龙虎榜详细数据，包括买卖营业部信息。

**参数：**
- `stock_code`: 股票代码，如 "002342"
- `date`: 日期，格式YYYY-MM-DD，默认为None（获取当日数据，16:30之后有效）
- `timeout`: 超时时间（秒），默认1600秒

**返回数据结构：**
```python
{
    "stock_code": "002342",         # 股票代码
    "stock_name": "巨力索具",        # 股票名称
    "date": "2026-02-11",           # 日期
    "current_price": "13.48",       # 当前价
    "change_pct": "10.04",          # 涨跌幅
    "turnover_ratio": "15.6",       # 换手率
    "circulating_shares": "5.2",    # 流通股本（亿股）
    "net_buy_amount": 50000000,     # 净买入金额
    "turnover": 500000000,          # 成交额
    "more_turnover": 0,             # 更多成交额
    "on_list_count": 10,            # 上榜次数
    "on_list_start": 20220315,      # 首次上榜
    "on_time_list": ["2026-02-11", "2026-02-10"],  # 历史上榜日期列表
    "buy_sell_data": [              # 买卖数据列表
        {
            "buy_list": [           # 买入营业部列表
                {
                    "seat_id": "80000001",          # 席位ID
                    "seat_name": "机构专用",         # 席位名称
                    "buy_amount": 30000000,         # 买入金额
                    "sell_amount": 0,               # 卖出金额
                    "rank": 1,                      # 排名
                    "group_id": "1",                # 分组ID
                    "group_icon": ["量化抢筹"]       # 分组标签
                }
            ],
            "sell_list": [          # 卖出营业部列表
                {
                    "seat_id": "80000002",
                    "seat_name": "某某营业部",
                    "buy_amount": 0,
                    "sell_amount": 20000000,
                    "rank": 1,
                    "group_id": "",
                    "group_icon": []
                }
            ],
            "up_reason": ["日涨幅偏离值达7%"],  # 上榜原因
            "buy_total": 100000000,             # 买入总额
            "sell_total": 80000000              # 卖出总额
        }
    ]
}
```

**使用示例：**
```python
from kaipanla_crawler import KaipanlaCrawler

crawler = KaipanlaCrawler()

# 获取当日龙虎榜详情（16:30之后有效）
result = crawler.get_longhubang_stock_detail("002342")

# 获取历史龙虎榜详情
result = crawler.get_longhubang_stock_detail("002342", "2026-02-10")

# 查看基本信息
print(f"股票: {result['stock_name']} ({result['stock_code']})")
print(f"涨跌幅: {result['change_pct']}%")
print(f"净买入: {result['net_buy_amount']:,.0f}")
print(f"上榜次数: {result['on_list_count']}")

# 查看买入营业部
for data in result['buy_sell_data']:
    print(f"\n上榜原因: {', '.join(data['up_reason'])}")
    print(f"买入总额: {data['buy_total']:,.0f}")
    print(f"卖出总额: {data['sell_total']:,.0f}")
    
    print("\n买入前五:")
    for seat in data['buy_list'][:5]:
        print(f"  {seat['seat_name']}: {seat['buy_amount']:,.0f}")
        if seat['group_icon']:
            print(f"    标签: {', '.join(seat['group_icon'])}")
    
    print("\n卖出前五:")
    for seat in data['sell_list'][:5]:
        print(f"  {seat['seat_name']}: {seat['sell_amount']:,.0f}")
```

### 3. get_longhubang_dataframe() - 获取DataFrame格式数据

获取龙虎榜上榜个股列表并返回pandas DataFrame格式，方便数据分析。

**参数：**
- `date`: 日期，格式YYYY-MM-DD，默认为None（获取当日数据）
- `timeout`: 超时时间（秒），默认1600秒

**返回：**
- pandas DataFrame，包含所有上榜个股的数据

**使用示例：**
```python
from kaipanla_crawler import KaipanlaCrawler

crawler = KaipanlaCrawler()

# 获取当日龙虎榜DataFrame
df = crawler.get_longhubang_dataframe()

# 获取历史龙虎榜DataFrame
df = crawler.get_longhubang_dataframe("2026-02-10")

# 数据分析
print(f"上榜个股数量: {len(df)}")
print(f"平均换手率: {df['turnover_ratio'].astype(float).mean():.2f}%")
print(f"总买入金额: {df['buy_amount'].sum():,.0f}")

# 筛选高换手率个股
high_turnover = df[df['turnover_ratio'].astype(float) > 20]
print(f"\n换手率>20%的个股:")
print(high_turnover[['stock_name', 'stock_code', 'change_pct', 'turnover_ratio']])
```

## 接口地址

- 龙虎榜列表: `https://applhb.longhuvip.com/w1/api/index.php`
- 龙虎榜详情: `https://applhb.longhuvip.com/w1/api/index.php`

## 注意事项

1. 当日龙虎榜数据通常在16:30之后才能获取
2. 历史数据可以随时获取
3. 分页参数 `page_size` 默认500条，一般足够获取当日所有上榜个股
4. 营业部标签（group_icon）可能包含"量化抢筹"、"游资"等标识
5. 机构席位通常以"机构专用"命名
