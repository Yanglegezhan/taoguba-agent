# 数据更新说明

## 为什么没有最新数据？

### 交易日说明

中国A股市场的交易时间：
- **交易日**: 周一至周五（法定节假日除外）
- **非交易日**: 周六、周日、法定节假日

例如：
- 2025-12-26（周五）- 交易日 ✓
- 2025-12-27（周六）- 非交易日 ✗
- 2025-12-28（周日）- 非交易日 ✗
- 2025-12-29（周一）- 交易日 ✓

### 数据更新时机

市场数据通常在**交易日收盘后**（15:00之后）才能获取到当天的完整数据。

如果你在交易日的：
- **15:00之前**: 只能获取到前一个交易日的数据
- **15:00之后**: 可以获取到当天的数据

## 如何更新数据？

### 方法1：一键更新（推荐）

进入 `batch/` 文件夹，双击运行：

```
一键更新.bat
```

这会自动：
1. 获取最新市场数据（需要10-20分钟）
2. 分析连板情况
3. 生成彩色溢价表

**注意**: 第一步获取数据需要较长时间，请耐心等待。

### 方法2：仅更新数据

如果只想更新数据，不生成报表：

```bash
cd 量化交易1/market_monitor
python scripts/main.py --mode update
```

### 方法3：检查数据日期

运行以下命令查看当前数据的日期范围：

```bash
cd 量化交易1/market_monitor
python scripts/check_dates.py
```

## 常见问题

### Q1: 为什么数据更新这么慢？

**A**: 系统需要从网络获取约5000只股票的数据，每只股票都需要单独请求，所以需要较长时间。

**优化建议**:
- 确保网络连接稳定
- 避免在网络高峰期更新
- 可以考虑使用更快的数据源（需要付费API）

### Q2: 数据更新失败怎么办？

**A**: 如果看到连接错误（Connection aborted），这是正常的，系统会自动跳过失败的股票继续获取其他数据。

**解决方法**:
1. 等待当前更新完成
2. 再次运行更新命令，补充失败的数据
3. 如果多次失败，检查网络连接

### Q3: 可以在交易时间更新数据吗？

**A**: 可以，但建议在收盘后（15:00之后）更新，这样能获取到完整的当天数据。

### Q4: 周末需要更新数据吗？

**A**: 不需要。周末不是交易日，没有新数据。建议在周一收盘后更新。

### Q5: 如何知道数据是否是最新的？

**A**: 运行检查脚本：

```bash
python scripts/check_dates.py
```

会显示：
- 数据的日期范围
- 最新数据日期
- 是否需要更新

## 自动化建议

### Windows任务计划程序

可以设置每个交易日15:30自动运行更新：

1. 打开"任务计划程序"
2. 创建基本任务
3. 触发器：每天15:30
4. 操作：运行 `batch/一键更新.bat`
5. 条件：仅在工作日运行

### Python定时任务

使用 `schedule` 库：

```python
import schedule
import time
from datetime import datetime

def is_trading_day():
    """判断是否是交易日"""
    today = datetime.now()
    # 简单判断：周一到周五
    return today.weekday() < 5

def update_data():
    """更新数据"""
    if is_trading_day():
        os.system('cd batch && 一键更新.bat')

# 每天15:30运行
schedule.every().day.at("15:30").do(update_data)

while True:
    schedule.run_pending()
    time.sleep(60)
```
