## 🚀 三步开始使用

### 第一步：安装依赖

进入 `batch/` 文件夹，双击运行：

```
安装依赖.bat
```

这会自动安装所有需要的Python库。

---

### 第二步：初始化数据（仅首次使用）

双击运行：

```
初始化数据.bat
```

这会获取最近2年的历史数据，大约需要5-10分钟。

---

### 第三步：生成溢价表

双击运行：

```
一键更新.bat
```

这会：
1. 获取最新市场数据
2. 分析连板情况
3. 生成彩色溢价表

完成后，打开 `output/连板溢价表_彩色版.xlsx` 查看结果！

---

## 📊 溢价表说明

### 表格内容

- **前7行**: 当日4板以上高度板（格式：股票名 连板数 题材）
- **主表**: 最近20天的连板溢价数据

### 列说明

| 列名 | 说明 |
|------|------|
| 日期 | 交易日期 |
| 股票代码 | 6位股票代码 |
| 股票名称 | 股票名称 |
| 连板数 | 连续涨停板数 |
| 涨跌幅 | 当日涨跌幅(%) |
| 概念 | 最相关的2个概念（用+连接） |

### 颜色说明

**连板颜色**（渐变红色）:
- 🌸 1-2板：浅粉色
- 🌺 3-4板：浅红色
- 🔴 5-6板：红色
- 🔥 7板+：深红色

**涨跌幅颜色**:
- 🟢 跌停：深绿底白字
- 🟢 大跌(-5%以下)：绿底白字
- 🟢 小跌：深绿字
- ⚪ 平盘：灰色字
- 🔴 小涨：红色字
- 🔴 大涨(5%以上)：红底白字

## 💡 智能概念匹配

### 工作原理

当股票首次2板时，系统会：

1. 获取该股票的所有概念
2. 计算每个概念在当天的板块平均涨幅
3. 选择涨幅最高的前2个概念作为最相关概念

### 优势

- ✅ 自动识别驱动题材
- ✅ 反映当天市场热点
- ✅ 提高选股准确性

## 🔧 高级功能

### 手动添加概念

如果某些股票的概念不准确，可以手动添加：

1. 打开 `scripts/manual_concepts.py`
2. 添加股票代码和概念：

```python
MANUAL_CONCEPTS = {
    '000001': '银行+金融科技',
    '600519': '白酒+消费',
}
```

3. 重新运行 `一键更新.bat`

### 获取更多概念数据

系统支持三种数据源：

1. **东方财富**（推荐，免费）
   ```
   双击: 获取概念_东财.bat
   ```

2. **同花顺**（补充数据源）
   ```
   双击: 获取概念_同花顺V2.bat
   ```

3. **Tushare**（需要token）
   ```
   双击: 获取概念_Tushare.bat
   ```

## ❓ 常见问题

### Q1: 批处理文件无法运行？

**A**: 确保在 `batch/` 文件夹中双击运行批处理文件。

### Q2: 提示缺少Python库？

**A**: 运行 `安装依赖.bat` 安装所有依赖。

### Q3: 数据获取失败？

**A**: 
- 检查网络连接
- 可能是数据源限流，等待几分钟后重试
- 尝试使用其他数据源

### Q4: Excel文件打不开？

**A**: 
- 确保安装了 Excel 2007 或更高版本
- 或使用 WPS Office
- 或查看CSV版本：`连板溢价表_最终版.csv`

### Q5: 概念数据不完整？

**A**: 运行多个数据源获取脚本：
```
获取概念_东财.bat
获取概念_同花顺V2.bat
```

## ✨ 使用技巧

### 1. 关注高度板

前7行显示的4板以上个股往往是市场热点，重点关注。

### 2. 观察概念轮动

通过连续几天的溢价表，可以观察到概念板块的轮动情况。

### 3. 结合涨跌幅

连板数高但涨跌幅低的股票可能面临压力，需要谨慎。

### 4. 定期更新概念

建议每周运行一次概念获取脚本，保持概念数据最新。
