| 3-4板 | 🌺 浅红色 | #FFB6C1 | 中级连板 |
| 5-6板 | 🔴 红色 | #FF6B6B | 高级连板 |
| 7板+ | 🔥 深红色 | #DC143C | 超高连板 |

### 2. 涨跌幅颜色（优化配色）

| 范围 | 颜色 | 十六进制 | 说明 |
|------|------|----------|------|
| 跌停 (≤-9.9%) | ⚫ 黑底白字 | #000000 | **醒目标识** |
| 大跌 (-9.9%~-5%) | 🟢 绿底白字 | #66BB6A | 大幅下跌 |
| 小跌 (-5%~0%) | 🟢 深绿字 | #388E3C | 小幅下跌 |
| 平盘 (=0%) | ⚪ 灰色字 | #757575 | 无涨跌 |
| 小涨 (0%~5%) | 🔴 红色字 | #D32F2F | 小幅上涨 |
| 大涨 (>5%) | 🔴 红底白字 | #EF5350 | 大幅上涨 |

### 3. 股票名称背景色（板块区分）⭐ 新增

用淡色背景区分不同板块的股票，**应用于所有日期列中的股票名称**：

| 板块 | 股票代码特征 | 背景色 | 十六进制 | 说明 |
|------|-------------|--------|----------|------|
| 主板 | 000/001/002/600/601/603 | 淡灰色 | #E8E8E8 | 沪深主板 |
| 科创板 | 688开头 | 淡红色 | #FFD6D6 | 科创板 |
| 创业板 | 300/301开头 | 淡蓝色 | #D6E4FF | 创业板 |
| 北交所 | 8/4开头 | 淡黄色 | #FFF4CC | 北交所 |

**注意**：
- 板块背景色应用于所有包含股票名称的单元格
- 包括第3列（股票名称列）和所有日期列中的股票名称
- 前7行的复合格式（如"金富科技 6板 AI+算力"）也会应用板块背景色

## 🎯 设计理念

### 1. 跌停黑底白字

**原因**:
- 跌停是重要的风险信号，需要特别醒目
- 黑色背景在Excel中非常显眼，一眼就能看到
- 与其他颜色形成强烈对比

**效果**:
- ⚫ 黑底白字 vs 🟢 绿底白字
- 跌停股票立即引起注意
- 便于快速识别风险

### 2. 板块淡色背景

**原因**:
- 不同板块有不同的交易规则和风险特征
- 科创板和创业板涨跌幅限制为20%
- 主板涨跌幅限制为10%
- 北交所有独特的交易规则

**效果**:
- 一眼识别股票所属板块
- 淡色背景不影响其他信息的阅读
- 便于快速评估风险

### 3. 渐变红色连板

**原因**:
- 连板数越高，市场关注度越高
- 颜色渐变直观反映连板强度
- 符合中国股市"红涨绿跌"的习惯

## 🔍 使用技巧

### 1. 快速识别跌停

在Excel中，黑底白字的单元格非常醒目：
- 打开溢价表
- 黑色单元格立即映入眼帘
- 这些是跌停股票，需要特别关注

### 2. 板块风险评估

根据股票名称的背景色：
- **淡灰色（主板）**: 相对稳健，涨跌幅10%
- **淡粉色（科创板）**: 高风险高收益，涨跌幅20%
- **淡蓝色（创业板）**: 高风险高收益，涨跌幅20%
- **淡黄色（北交所）**: 流动性较低，需谨慎

### 3. 连板强度判断

根据颜色深浅：
- **浅粉色（1-2板）**: 初级连板，观察阶段
- **浅红色（3-4板）**: 中级连板，关注阶段
- **红色（5-6板）**: 高级连板，谨慎阶段
- **深红色（7板+）**: 超高连板，高风险高收益

## 🎨 颜色选择原则

### 1. 对比度

- 黑底白字：最高对比度，最醒目
- 绿底白字：高对比度，清晰可读
- 红底白字：高对比度，清晰可读

### 2. 淡色背景

- 淡灰色：中性，不抢眼
- 淡粉色：温和，易识别
- 淡蓝色：清爽，易识别
- 淡黄色：温暖，易识别

### 3. Material Design

所有颜色均参考Google Material Design规范：
- 色彩饱和度适中
- 对比度符合可访问性标准
- 视觉舒适，不刺眼

## 🔧 技术实现

### 跌停判断

```python
if pct <= -9.9:  # 跌停
    return ('404040', 'FFFFFF')  # 灰黑底白字（醒目但不刺眼）
```

### 板块判断

```python
def get_board_bg_color(stock_code):
    code_str = str(stock_code).zfill(6)
    
    if code_str.startswith('688'):
        return 'FFD6D6'  # 科创板 - 淡红色
    elif code_str.startswith('300') or code_str.startswith('301'):
        return 'D6E4FF'  # 创业板 - 淡蓝色
    elif code_str.startswith('8') or code_str.startswith('4'):
        return 'FFF4CC'  # 北交所 - 淡黄色
    else:
        return 'E8E8E8'  # 主板 - 淡灰色
```

### 股票名称识别

```python
# 直接匹配股票名称
if cell_value in stock_name_to_code:
    apply_board_color()

# 从复合字符串中提取股票名称（如"金富科技 6板 AI+算力"）
for stock_name in stock_name_to_code.keys():
    if cell_value.startswith(stock_name + ' '):
        apply_board_color()
```

## ✅ 总结

v3.1版本的配色方案在v3.0的基础上进行了两项重要改进：

1. **跌停灰黑底白字**: 使用灰黑色(#404040)而非纯黑色，更醒目但不刺眼，更容易识别风险
2. **股票名称板块背景色**: 新增功能，所有包含股票名称的单元格都会根据板块显示不同的淡色背景
   - 应用于第3列（股票名称列）
   - 应用于所有日期列中的股票名称
   - 应用于前7行的复合格式（如"金富科技 6板 AI+算力"）

这些改进使得溢价表更加实用，信息更加丰富，风险识别更加快速。
