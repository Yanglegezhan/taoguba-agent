- ✅ 覆盖所有2板及以上的股票（100%）

- ✅ 覆盖断板后重新连板的股票（100%）

### 3. 智能识别

- 自动获取最近20天的涨停股票
- 自动去重（同一股票多次涨停只计一次）
- 自动合并到历史数据

### 4. 容错机制

- 如果智能更新失败，自动切换到全量更新
- 保留全量更新选项，以备不时之需

## 🔧 技术实现

### 算法流程

```
开始
  ↓
获取最近20天的涨停股票列表
  ↓
去重（547只股票）
  ↓
只获取这些股票的最近5天数据
  ↓
合并到历史数据
  ↓
去重（保留最新数据）
  ↓
保存
  ↓
完成
```

### 关键代码

```python
# 1. 获取涨停股票列表
limit_up_stocks = self.data_fetcher.get_recent_limit_up_stocks(days=20)

# 2. 只获取这些股票的数据
new_data = self.data_fetcher.get_selected_stocks_data(
    limit_up_stocks, 
    start_date_str, 
    end_date_str
)

# 3. 合并并去重
combined_data = pd.concat([old_data, new_data], ignore_index=True)
combined_data = combined_data.drop_duplicates(
    subset=['股票代码', '日期'], 
    keep='last'
)
```
