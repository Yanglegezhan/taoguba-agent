# 连板溢价表生成系统

A股市场连板监控与概念分析系统 - 自动生成连板溢价表

## 🚀 快速开始

### 1. 安装依赖
```bash
双击运行: 安装依赖.bat
```

### 2. 初始化数据
```bash
双击运行: 初始化数据.bat
```

### 3. 获取概念(首次使用)
```bash
双击运行: 获取概念_Tushare.bat    # 推荐,数据量大(4000+)
或者:     获取概念_东财.bat        # 备选,速度快(3-5分钟)
```

### 4. 生成表格
```bash
双击运行: 一键更新.bat             # 完整流程
或者:     仅生成报表.bat           # 仅生成表格
```

## 📊 输出文件

- `output/连板溢价表_最终版.csv` - 最终表格
- `output/board_analysis.csv` - 连板分析数据
- `output/concept_cache.csv` - 概念缓存

## 📈 表格说明

### 表格结构
- 第1列: 序号
- 第2列: 概念(格式: "概念1+概念2")
- 第3列: 股票名称(仅第一个交易日的股票)
- 第4列起: 日期数据(20个交易日)

### 数据规则
- 前7行: 当日4板以上个股
- 第8行起: 2板以上个股追踪
- 断板后继续追踪(显示涨跌幅)
- 断板后重新2板会在新行显示

## 📞 常见问题

### Q1: 如何提高概念覆盖率?
A: 1. 使用Tushare获取(覆盖率90%+)

A: 1. 检查是否已运行初始化数据
   2. 检查output目录是否存在
   3. 查看错误信息
