## 📊 主要功能

### 1. 连板溢价表

**文件**: `scripts/premium_table_v3_colored.py`

**功能**:
- 生成最近20天的连板溢价表
- 彩色标记（连板渐变红色，涨跌幅多级颜色）
- 智能概念关联（基于2板当天概念板块涨幅）
- 前7行显示当日4板以上个股

**输出**:
- `output/连板溢价表_彩色版.xlsx` - Excel彩色版
- `output/连板溢价表_最终版.csv` - CSV纯数据版

---

### 2. 概念获取

**支持的数据源**:
- Tushare (推荐)
- 东方财富
- 同花顺

**批处理文件**:
- `batch/获取概念_Tushare.bat`
- `batch/获取概念_东财.bat`

## 🎨 彩色溢价表说明

### 连板颜色

| 连板数 | 颜色 |
|--------|------|
| 1-2板 | 🌸 浅粉色 |
| 3-4板 | 🌺 浅红色 |
| 5-6板 | 🔴 红色 |
| 7板+ | 🔥 深红色 |

### 涨跌幅颜色

| 范围 | 颜色 |
|------|------|
| 跌停 (≤-9.9%) | ⚫ 灰黑底白字 |
| 大跌 (-9.9%~-5%) | 🟢 绿底白字 |
| 小跌 (-5%~0%) | 🟢 深绿字 |
| 平盘 (=0%) | ⚪ 灰色字 |
| 小涨 (0%~5%) | 🔴 红色字 |
| 大涨 (>5%) | 🔴 红底白字 |

### 股票名称背景色（板块区分）⭐ 新增

| 板块 | 背景色 |
|------|--------|
| 主板 | 淡灰色 |
| 科创板(688) | 淡红色 |
| 创业板(300/301) | 淡蓝色 |
| 北交所(8/4开头) | 淡黄色 |

**说明**: 所有日期列中的股票名称都会根据板块显示不同的淡色背景，便于快速识别股票所属板块。

## 💡 智能概念关联

**原理**: 当股票首次2板时，计算其所有概念在当天的板块平均涨幅，选择涨幅最高的前2个作为最相关概念。

**优势**:
- ✅ 自动识别驱动题材
- ✅ 反映当天市场热点
- ✅ 提高选股准确性

## 🐛 常见问题

### Q1: 批处理文件无法运行？

**A**: 确保在 `batch/` 文件夹中运行，或使用绝对路径。

### Q2: 概念数据不完整？

**A**: 运行多个数据源获取脚本：
```bash
batch/获取概念_Tushare.bat
batch/获取概念_东财.bat
batch/获取概念_同花顺V2.bat
```

### Q3: Excel文件打不开？

**A**: 确保安装了 Excel 2007 或更高版本，或使用 WPS Office。
