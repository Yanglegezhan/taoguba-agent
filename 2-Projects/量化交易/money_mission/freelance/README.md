## 接单策略
1. **快速接单** - 优先接小而快的项目，快速积累资金
2. **口碑积累** - 保证交付质量，争取复购
3. **主动投标** - 在平台主动搜索和投标
4. **产品化** - 将常用功能产品化，提高效率

## 注意事项
1. 仔细阅读需求，避免理解偏差
2. 明确交付时间和功能范围
3. 预收定款或使用平台担保
4. 保留开发过程记录，避免纠纷
5. 及时沟通，让客户了解进度
