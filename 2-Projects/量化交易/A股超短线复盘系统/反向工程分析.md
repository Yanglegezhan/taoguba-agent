## 报告整体结构

### 固定7个章节（所有报告一致）

```
一、大盘总结
二、板块与主线分析
三、情绪周期判断（重点）
四、主线与支线详细梳理
五、昨日高标PK分析
六、首板个股逻辑总结（按题材分类）
七、明日策略与风险提示
```

### 每个报告包含的关键元素

- ✅ **定性判断** - 大盘总结中的总结性判断
- ✅ **表格分析** - 板块强度对比表格
- ✅ **高亮框** - success-box/warning-box/info-box
- ✅ **emoji标记** - 📊✅⚠️🔥🎯等
- ✅ **数据高亮** - `<span class="data-highlight">`
- ✅ **策略口诀** - 最后的总结性口诀

## 各章节分析方法

### 一、大盘总结

**分析维度：**
1. **指数走势**
   - 格式：`昨日[涨跌]，今日[涨跌]，[描述走势形态]`
   - 数据：上证/深证/中证全指涨跌幅
   - 形态判断："V型反转"、"高开高走"、"低开低走"等

2. **量能变化**
   - 数据：成交额（万亿级别）
   - 判断：是否维持高位、资金是否离场

3. **市场情绪**
   - 上涨/下跌家数及比例
   - 涨停/跌停家数
   - 情绪转变（谨慎→积极、积极→谨慎）

**输出要求：**
- 必须包含**定性判断**段落
- 格式：`<div class="success-box">✅ 定性判断：[判断内容]</div>`

---

### 二、板块与主线分析

**分析维度：**

1. **强势板块及原因**（表格形式）
   | 板块 | 涨幅 | 涨停贡献 | 核心逻辑 |
   |---|---|---|---|

2. **资金流向判断**
   - 流入板块：科技成长赛道、低位题材
   - 流出板块：周期题材、高位兑现题材
   - 判断依据：板块涨幅、涨停股数量、概念关联

3. **与涨停股对应性验证**
   - 板块内涨停股与涨幅的关系
   - 封板时间（早盘/午盘/尾盘）
   - 梯队完整性

**输出要求：**
- 必须包含**结论**段落
- 格式：`<div class="success-box">✅ [结论]</div>`

---

### 三、情绪周期判断（重点）

**分析维度：**

1. **昨日高位股表现分析**
   - 昨日3板及以上个股数量
   - 今日晋级成功数量
   - **晋级率 = 成功数/总数**
   - 判断标准：14.3% < 20%警戒线 → 极低水平

2. **关键信号识别**
   - 是否出现核按钮（跌停）
   - 高位股是否集体补跌
   - 是否有独立逻辑个股（如ST股）

3. **资金行为解读**
   - 高位是否兑现
   - 资金切换方向（高低切）
   - 新旧周期交替判断

**输出要求：**
- 必须包含**情绪周期定位**段落
- 必须包含**风险偏好**段落
- 必须包含**明日策略建议**段落

---

### 四、主线与支线详细梳理

**主线分析框架：**

```
🔥 主线1：[题材名称]
高度龙：[个股]（[板数]，[逻辑简述]）
中军：[个股1]、[个股2]、[个股3]
首板梯队：[列举个股]
板块强度：
- [数量统计]
- [技术分析]
- [逻辑评价]

✅ 支撑数据：[数据支撑]
```

**支线分析框架：**

```
🌿 支线：[题材名称]
表现个股：[列举]
特征：[特征描述]
风险：[风险提示]
```

**分类标准：**
- **主线**：涨停数量多、概念逻辑硬核、资金集中度高
- **支线**：涨停数量少、事件驱动、缺乏持续性

---

### 五、昨日高标PK分析

**分析维度：**

1. **2进3晋级情况**
   - 昨日2板及以上数量
   - 今日晋级3板及以上数量
   - 晋级率计算

2. **失败代表分析**
   - 失败个股及其原因
   - 是否有核按钮
   - 缺乏哪些特征

3. **胜出特征总结**
   - 成功个股的共同特征
   - 政策背书
   - 概念叠加
   - 封板时间

4. **1进2晋级情况**
   - 昨日首板数量（可估算）
   - 今日2板数量
   - 成功率评价

---

### 六、首板个股逻辑总结（按题材分类）

**分类方式：**
1. 新型显示
2. 新型电力系统
3. 算力与数据要素
4. 其他题材

**个股逻辑格式：**
```
- **个股名称**：[核心逻辑1]+[核心逻辑2]+[核心逻辑3]，[受益点]
```

**逻辑要素：**
- 龙头地位（全球龙头、行业第一）
- 技术优势（量产、突破、送样）
- 订单落地（中标、签订合同）
- 政策受益（国企改革、新型工业化）
- 概念叠加（多题材）

---

### 七、明日策略与风险提示

**机会方向：**
- 聚焦主线（主线名称+具体标的）
- 高低切机会（选股标准）

**风险警示：**
- 高位股风险（断板、补跌）
- 题材退潮信号（失败指标）
- 流动性风险（资金集中）

**总策略口诀：**
格式：`<div class="highlight-box">🎯 <strong>总策略口诀：</strong><br><strong>[口诀内容]</strong></div>`

### 核心计算指标

```python
# 晋级率计算
晋级率 = 昨日高位股今日晋级成功数 / 昨日高位股总数

# 高低切判断
如果 高位股晋级率 < 20% 且 低位题材涨停数增加：
    判定为高低切

# 情绪周期定位
如果 大盘系数 < 30 且 情绪指标 < 50 且 亏钱效应 > 100:
    冰点期
elif 晋级率 < 20% 且 出现核按钮:
    退潮末期
elif 涨停数量回升 且 低位题材活跃:
    主升初期
```

| 特征 | 主线 | 支线 |

| 逻辑 | 硬核（技术/政策） | 事件驱动 |
| 资金 | 持续性 | 一次性 |

   def identify_main_theme(sectors):

3. **首板股逻辑提取：**
   ```python
   def extract_logic(stock):
       # 从概念标签提取核心逻辑
       concepts = stock['concepts'].split('、')
       logic = []
       for concept in concepts:
           if '龙头' in concept:
               logic.append(f"{stock['name']}是{concept}")
           elif '量产' in concept or '突破' in concept:
               logic.append(f"{concept}落地")
           elif '政策' in concept or '改革' in concept:
               logic.append(f"{concept}催化")
       return ' + '.join(logic)
   ```
